    1.1 Java语言概述

    Java是一种广泛使用的高级编程语言，由Sun Microsystems于1995年发布。Java具有跨平台、面向对象、安全性高等特点，被广泛应用于企业级开发、移动应用（Android）、大数据处理等领域。

    Java的核心优势：

    • 跨平台性：一次编写，随处运行（Write Once, Run Anywhere）
    • 面向对象：支持封装、继承、多态等OOP特性
    • 自动内存管理：垃圾回收机制
    • 安全性：内置安全机制

    1.2 JDK安装与环境配置

    JDK（Java Development Kit）是Java开发环境的核心组件，包含编译器、调试器等工具。

    1. 访问Oracle官网下载JDK
    2. 安装JDK到指定目录
    3. 配置环境变量：
      • Windows：设置JAVA_HOME和Path
      • Linux/Mac：配置~/.bashrc或~/.zshrc
    4. 验证安装：在终端输入java -version和javac -version

    1.3 编写第一个Java程序

    创建一个简单的Hello World程序：

    public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

    编译与运行

    # 编译
javac HelloWorld.java

# 运行
java HelloWorld

# 输出
Hello, World!

    1.4 常见错误示例

    ❌ 错误1：类名与文件名不一致
    如果文件名为Hello.java，但类名为HelloWorld，编译会失败。
    解决方案：确保类名与文件名完全一致（包括大小写）。
    ❌ 错误2：缺少main方法
    Java程序需要一个main方法作为入口点。
    解决方案：添加public static void main(String[] args)方法。
    ❌ 错误3：未配置环境变量
    运行javac时提示"不是内部或外部命令"。
    解决方案：正确配置JAVA_HOME和Path环境变量。

    2.1 数据类型

    Java是强类型语言，每个变量必须声明其类型。Java的数据类型分为两大类：

    • 基本类型（Primitive Types）：byte, short, int, long, float, double, boolean, char
    • 引用类型（Reference Types）：类、接口、数组等
    public class DataTypes {
    public static void main(String[] args) {
        // 整数类型
        byte b = 100;
        short s = 32000;
        int i = 1000000;
        long l = 10000000000L;
        
        // 浮点类型
        float f = 3.14f;
        double d = 3.1415926;
        
        // 字符和布尔
        char c = 'A';
        boolean flag = true;
    }
}

    2.2 变量与常量

    变量是存储数据的容器，常量是不可变的值。

    public class Variables {
    public static void main(String[] args) {
        // 变量声明与赋值
        int age = 25;
        String name = "张三";
        
        // 常量（final关键字）
        final double PI = 3.14159;
        
        // 变量修改
        age = 26;
    }
}

    2.3 运算符

    Java支持多种运算符：

    • 算术运算符：+ - * / %
    • 赋值运算符：= += -= *= /= %=
    • 比较运算符：== != > < >= <=
    • 逻辑运算符：&& || !
    • 位运算符：& | ^ ~ << >> >>>
    public class Operators {
    public static void main(String[] args) {
        int a = 10, b = 3;
        
        System.out.println(a + b); // 13
        System.out.println(a - b); // 7
        System.out.println(a * b); // 30
        System.out.println(a / b); // 3（整数除法）
        System.out.println(a % b); // 1（取模）
        
        // 自增自减
        System.out.println(a++); // 10，然后a=11
        System.out.println(++a); // 12
    }
}

    2.4 类型转换

    Java中的类型转换分为隐式转换（自动转换）和显式转换（强制转换）。

    public class TypeConversion {
    public static void main(String[] args) {
        // 隐式转换：小类型转大类型
        int num = 100;
        double decimal = num; // 自动转换
        
        // 显式转换：大类型转小类型
        double pi = 3.14;
        int approx = (int) pi; // 强制转换，approx = 3
    }
}

    2.5 常见错误示例

    ❌ 错误1：变量未初始化
    Java要求变量在使用前必须初始化。
    int x; System.out.println(x); // 编译错误
    ❌ 错误2：类型不匹配
    不能直接将double赋值给int。
    int a = 3.14; // 编译错误，需要强制转换
    ❌ 错误3：整数除法
    两个整数相除结果仍是整数，小数部分被丢弃。
    System.out.println(5 / 2); // 输出2，不是2.5
    解决方案：将其中一个数转为double。

    3.1 条件语句

    if-else语句用于根据条件执行不同的代码块。

    public class IfElse {
    public static void main(String[] args) {
        int score = 85;
        
        if (score >= 90) {
            System.out.println("优秀");
        } else if (score >= 80) {
            System.out.println("良好");
        } else if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }
    }
}

    3.2 分支语句

    switch语句用于多条件分支，Java 7+支持String类型。

    public class SwitchCase {
    public static void main(String[] args) {
        String day = "Monday";
        
        switch (day) {
            case "Monday":
            case "Tuesday":
            case "Wednesday":
            case "Thursday":
            case "Friday":
                System.out.println("工作日");
                break;
            case "Saturday":
            case "Sunday":
                System.out.println("周末");
                break;
            default:
                System.out.println("无效日期");
        }
    }
}

    3.3 循环语句

    Java提供三种循环结构：for、while和do-while。

    public class Loops {
    public static void main(String[] args) {
        // for循环
        for (int i = 1; i <= 5; i++) {
            System.out.println("for: " + i);
        }
        
        // while循环
        int j = 1;
        while (j <= 5) {
            System.out.println("while: " + j);
            j++;
        }
        
        // do-while循环（至少执行一次）
        int k = 1;
        do {
            System.out.println("do-while: " + k);
            k++;
        } while (k <= 5);
    }
}

    3.4 跳转语句

    break、continue和return用于控制程序流程。

    public class JumpStatements {
    public static void main(String[] args) {
        // break：跳出循环
        for (int i = 1; i <= 10; i++) {
            if (i == 5) break;
            System.out.println(i); // 输出1-4
        }
        
        // continue：跳过当前循环
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) continue;
            System.out.println(i); // 输出奇数
        }
    }
}

    3.5 常见错误示例

    ❌ 错误1：switch语句中缺少break
    忘记break会导致"fall through"，多个case会被执行。
    解决方案：在每个case末尾添加break。
    ❌ 错误2：死循环
    循环条件永远为true会导致程序无限运行。
    while (true) { ... } // 除非内部有break
    ❌ 错误3：for循环中初始化变量类型错误
    for (int i = 0; i < 10; i++) { ... }是正确的，
    for (i = 0; i < 10; i++) { ... }需要提前声明i。

    4.1 数组

    数组是存储相同类型元素的固定大小容器。

    public class ArrayExample {
    public static void main(String[] args) {
        // 声明数组
        int[] numbers = new int[5];
        
        // 初始化数组
        numbers[0] = 10;
        numbers[1] = 20;
        numbers[2] = 30;
        numbers[3] = 40;
        numbers[4] = 50;
        
        // 简化初始化
        int[] scores = {85, 90, 78, 92};
        
        // 遍历数组
        for (int i = 0; i < scores.length; i++) {
            System.out.println(scores[i]);
        }
        
        // 增强for循环
        for (int score : scores) {
            System.out.println(score);
        }
    }
}

    4.2 二维数组

    二维数组可以看作是数组的数组，常用于表示表格数据。

    public class TwoDArray {
    public static void main(String[] args) {
        // 创建3x3的二维数组
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };
        
        // 遍历二维数组
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
    }
}

    4.3 ArrayList

    ArrayList是Java集合框架中最常用的动态数组实现。

    import java.util.ArrayList;
import java.util.List;

public class ArrayListExample {
    public static void main(String[] args) {
        List<String> fruits = new ArrayList<>();
        
        // 添加元素
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");
        
        // 获取元素
        System.out.println(fruits.get(0)); // Apple
        
        // 修改元素
        fruits.set(1, "Grape");
        
        // 删除元素
        fruits.remove(2);
        
        // 遍历
        for (String fruit : fruits) {
            System.out.println(fruit);
        }
    }
}

    4.4 HashMap

    HashMap是基于哈希表的键值对集合。

    import java.util.HashMap;
import java.util.Map;

public class HashMapExample {
    public static void main(String[] args) {
        Map<String, Integer> ages = new HashMap<>();
        
        // 添加键值对
        ages.put("Alice", 25);
        ages.put("Bob", 30);
        ages.put("Charlie", 35);
        
        // 获取值
        System.out.println(ages.get("Bob")); // 30
        
        // 遍历
        for (Map.Entry<String, Integer> entry : ages.entrySet()) {
            System.out.println(entry.getKey() + ": " + entry.getValue());
        }
        
        // 检查键是否存在
        if (ages.containsKey("Alice")) {
            System.out.println("Alice exists");
        }
    }
}

    4.5 常见错误示例

    ❌ 错误1：数组越界异常
    访问超出数组长度的索引会抛出ArrayIndexOutOfBoundsException。
    int[] arr = new int[5]; arr[5] = 10; // 错误
    ❌ 错误2：使用原始类型ArrayList
    不指定泛型类型会导致类型不安全。
    ArrayList list = new ArrayList(); // 不推荐
    解决方案：使用ArrayList<String> list = new ArrayList<>();
