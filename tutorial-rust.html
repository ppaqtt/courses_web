    1. 入门介绍与环境搭建

    🌟 Rust的特点

    Rust是一种系统级编程语言，由Mozilla开发，于2010年首次发布。它以内存安全、零成本抽象和并发安全著称。

    • 内存安全：通过所有权系统保证内存安全，无需垃圾回收
    • 零成本抽象：高级抽象不带来运行时开销
    • 并发安全：编译时检查数据竞争
    • 高性能：与C/C++相当的性能
    • 跨平台：支持Windows、Mac、Linux等多种平台

    3. 所有权系统

    🔑 所有权规则

    • 每个值都有一个所有者
    • 值在任一时刻只能有一个所有者
    • 当所有者离开作用域，值将被丢弃
    rust 
    fn main() {
    let s1 = String::from("hello");
    let s2 = s1;
    
    // println!("{}", s1); // 错误！s1的所有权已移动到s2
    println!("{}", s2); // 正确
    
    let s3 = String::from("world");
    let s4 = s3.clone();
    
    println!("{} {}", s3, s4); // 正确，clone创建了副本
}

    🔗 借用与引用

    借用允许你引用某些值而不获取其所有权。

    rust 
    fn calculate_length(s: &String) -> usize {
    s.len()
}

fn main() {
    let s1 = String::from("hello");
    let len = calculate_length(&s1);
    println!("'{}' 的长度是 {}", s1, len);
    
    let mut s2 = String::from("hello");
    change(&mut s2);
    println!("s2 = {}", s2);
}

fn change(s: &mut String) {
    s.push_str(", world");
}

    ⏱️ 生命周期

    生命周期确保引用在使用期间始终有效。

    rust 
    fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}

fn main() {
    let string1 = String::from("abcd");
    let string2 = "xyz";
    
    let result = longest(string1.as_str(), string2);
    println!("最长的字符串是 {}", result);
}

    ❌ 常见错误

    使用已移动的值：let s1 = s2; println!("{}", s2); 会报错。

    可变引用与不可变引用冲突：同一时间只能有一个可变引用。

    💡 闭包（Closure）

    闭包是可以捕获其环境的匿名函数。

    rust 
    let add_one = |x: i32| -> i32 { x + 1 };
let add_two = |x| x + 2;

let x = 10;
let multiply = |y| x * y;
println!("10 * 5 = {}", multiply(5));

fn apply(f: F, value: i32) -> i32
where
    F: Fn(i32) -> i32,
{
    f(value)
}

fn main() {
    let result = apply(add_one, 5);
    println!("result = {}", result);
}

    ❌ 常见错误

    🔵 Result类型

    Result 用于可恢复的错误。

    rust 
    use std::fs::File;

fn main() {
    let file = File::open("hello.txt");
    let file = match file {
        Ok(f) => f,
        Err(e) => {
            println!("打开文件失败: {}", e);
            return;
        }

    ⚡ ? 操作符

    ? 操作符是传播错误的便捷方式。

    rust 
    use std::fs::File;
use std::io::{self, Read};

fn read_username_from_file() -> Result {
    let mut f = File::open("username.txt")?;
    let mut s = String::new();
    f.read_to_string(&mut s)?;
    Ok(s)
}

fn main() -> Result<(), Box> {
    let username = read_username_from_file()?;
    println!("username: {}", username);
    Ok(())
}

    ❌ 常见错误

    在main函数中使用?：需要将返回类型改为 Result<(), E>。

    过度使用unwrap()：会导致程序在错误时直接panic。

    🎯 项目概述

    创建一个简单的命令行待办事项应用，包含添加、删除、标记完成等功能。

    📦 添加依赖

    toml 
    [dependencies]
serde = { version = "1.0", features = ["derive"] }
serde_json = "1.0"
structopt = "0.3"

    🔧 实现代码

    rust 
    use serde::{Deserialize, Serialize};
use std::fs;
use structopt::StructOpt;

#[derive(Debug, StructOpt)]
enum Command {
    Add { task: String },
    List,
    Done { index: usize },
    Remove { index: usize },
}

#[derive(Debug, Serialize, Deserialize)]
struct Todo {
    tasks: Vec,
}

#[derive(Debug, Serialize, Deserialize)]
struct Task {
    description: String,
    completed: bool,
}

impl Todo {
    fn load() -> Self {
        if std::path::Path::new("todo.json").exists() {
            let content = fs::read_to_string("todo.json").unwrap();
            serde_json::from_str(&content).unwrap()
        } else {
            Todo { tasks: Vec::new() }
        }
    }
    
    fn save(&self) {
        let content = serde_json::to_string_pretty(self).unwrap();
        fs::write("todo.json", content).unwrap();
    }
    
    fn add(&mut self, description: String) {
        self.tasks.push(Task { description, completed: false });
    }
    
    fn list(&self) {
        for (i, task) in self.tasks.iter().enumerate() {
            let status = if task.completed { "✓" } else { " " };
            println!("{} [{}] {}", i + 1, status, task.description);
        }
    }
    
    fn done(&mut self, index: usize) {
        if let Some(task) = self.tasks.get_mut(index) {
            task.completed = true;
        }
    }
    
    fn remove(&mut self, index: usize) {
        self.tasks.remove(index);
    }
}

fn main() {
    let command = Command::from_args();
    let mut todo = Todo::load();
    
    match command {
        Command::Add { task } => {
            todo.add(task);
            println!("添加成功");
        }
        Command::List => todo.list(),
        Command::Done { index } => {
            todo.done(index - 1);
            println!("标记完成");
        }
        Command::Remove { index } => {
            todo.remove(index - 1);
            println!("删除成功");
        }
    }
    
    todo.save();
}
