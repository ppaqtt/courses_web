01 PHP简介与环境搭建

▼
知识点
  • PHP是一种服务器端脚本语言，用于Web开发
  • PHP文件以.php扩展名，可嵌入HTML中
  • 常用环境：WAMP(Linux)、XAMPP、MAMP
  • PHP标记：<?php ?> 或 <?= ?>
  • PHP执行流程：浏览器请求→服务器解析→返回HTML
代码示例
hello.php 
<!DOCTYPE html>
<html>
<body>
    <h1>我的第一个PHP页面</h1>
    <?php
        echo "Hello, PHP!";
        // 这是单行注释
        /* 这是多行注释 */
    ?>
    <p><?="短标记输出" ?></p>
</body>
</html>
常见错误
  • 忘记添加PHP结束标记 ?>
  • 在纯PHP文件末尾添加?>，可能导致输出空白
  • 使用短标记<? ?>需要在php.ini中开启
  • PHP文件未正确配置服务器，被当作文本下载
练习题
  1. 搭建PHP开发环境，确保能运行PHP脚本
  2. 创建一个PHP文件，输出"Hello, World!"
  3. 在PHP文件中混合HTML和PHP代码，输出当前日期
  4. 查看phpinfo()输出，了解当前PHP配置

04 常量

▼
知识点
  • 常量一旦定义不可更改
  • 使用define()或const定义
  • 常量名通常大写，不使用$符号
  • 常量在整个脚本中全局可用
  • 预定义常量：__FILE__, __LINE__, PHP_VERSION等
代码示例
constants.php 
<?php
    // 使用define定义常量
    define("PI", 3.14159);
    define("SITE_NAME", "我的网站");
    
    // 使用const定义常量（类外部也可用）
    const MAX_USERS = 1000;
    
    // 使用常量
    echo "圆周率：" . PI;
    echo "<br>网站名称：" . SITE_NAME;
    
    // 预定义常量
    echo "<br>当前文件：" . __FILE__;
    echo "<br>当前行号：" . __LINE__;
    echo "<br>PHP版本：" . PHP_VERSION;
    
    // 检查常量是否定义
    if (defined("PI")) {
        echo "<br>PI已定义";
    }
?>
常见错误
  • 尝试修改已定义的常量
  • 常量名使用小写（虽不报错但不符合规范）
  • 使用const在条件语句中定义常量（PHP 7.0+允许）
  • 混淆常量和变量的定义方式

05 运算符

▼
知识点
  • 算术运算符：+ - * / % **
  • 赋值运算符：= += -= *= /= %=
  • 比较运算符：== === != !== > < >= <=
  • 逻辑运算符：&& || ! and or xor
  • 递增/递减：++ --（前置和后置）
代码示例
operators.php 
<?php
    // 算术运算
    $a = 10;
    $b = 3;
    echo "加法：" . ($a + $b);
    echo "<br>取模：" . ($a % $b);
    echo "<br>幂运算：" . ($a ** $b);

    echo "<br>==：" . (10 == "10"); // true
    echo "<br>===：" . (10 === "10"); // false
    
    // 递增递减
    $x = 5;
    echo "<br>后置++：" . $x++; // 5
    echo "<br>前置++：" . ++$x; // 7
    
    // 逻辑运算
    if ($a > 5 && $b < 10) {
        echo "<br>条件成立";
    }
?>
常见错误
  • 使用==而非===进行严格比较
  • 除法运算结果为浮点数而非整数
  • 递增递减运算符的前置后置混淆
  • 逻辑运算符优先级错误

知识点

  • switch语句：等值判断
  • 三元运算符：简洁条件表达式
代码示例
conditionals.php 
<?php
    $score = 85;
    
    // if-elseif-else
    if ($score >= 90) {
        echo "优秀";
    } elseif ($score >= 80) {
        echo "良好";
    } elseif ($score >= 60) {
        echo "及格";
    } else {
        echo "不及格";

    $day = 3;

            echo "星期一";

        case 3:

            break;
        default:
            echo "其他";
    }

  • 条件判断中使用赋值=而非比较==

  • 三元运算符嵌套过多影响可读性

07 流程控制-循环语句

▼
知识点
  • for循环：已知次数的循环
  • while循环：条件满足时循环
  • do-while循环：至少执行一次
  • foreach循环：遍历数组
  • break和continue控制循环
代码示例
loops.php 
<?php
    // for循环
    for ($i = 1; $i <= 10; $i++) {
        echo $i . " ";
    }
    
    // while循环
    $count = 0;
    while ($count < 5) {
        echo "计数：" . $count . "<br>";
        $count++;
    }
    
    // do-while循环
    $x = 0;
    do {
        echo $x;
        $x++;
    } while ($x < 3);
    
    // foreach循环
    $fruits = ["苹果", "香蕉", "橙子"];
    foreach ($fruits as $fruit) {
        echo $fruit . "<br>";
    }
    
    // break和continue
    for ($i = 1; $i <= 10; $i++) {
        if ($i == 5) continue;
        if ($i > 8) break;
        echo $i;
    }
?>
常见错误
  • while循环中忘记更新计数器导致死循环
  • for循环条件错误导致循环不执行或死循环
  • foreach中直接修改数组元素
  • 混淆break和continue的作用
练习题
  1. 使用for循环输出1-100
  2. 使用while循环计算1-100的和
  3. 使用foreach遍历关联数组
  4. 练习break和continue的使用

08 函数基础

▼
知识点
  • 使用function关键字定义函数
  • 函数名不区分大小写，但建议一致
  • 函数必须先定义后调用
  • 函数内部有自己的作用域
  • 使用return返回值
代码示例
functions.php 
<?php
    // 定义无参数无返回值函数
    function greet() {
        echo "Hello, World!";
    }
    greet();
    
    // 定义有参数有返回值函数
    function add($a, $b) {
        return $a + $b;
    }
    $result = add(3, 5);
    echo $result;
    
    // 函数内部作用域
    $globalVar = "全局变量";

        $localVar = "局部变量";
        global $globalVar;
        echo $globalVar;
    }
    testScope();
    
    // 检查函数是否存在
    if (function_exists("greet")) {
        greet();
    }
?>
常见错误
  • 函数调用在定义之前
  • 函数名重复定义
  • 忘记使用global关键字访问全局变量
  • return语句后还有代码不会执行

09 函数参数与返回值

▼
知识点
  • 参数可以设置默认值
  • 参数可以是按值传递或引用传递
  • PHP 7+支持类型声明
  • 可以返回多个值（通过数组）
  • void返回类型（PHP 7.1+）
代码示例
function-params.php 
<?php
    // 默认参数值
    function sayHello($name = "Guest") {
        echo "Hello, $name!";
    }
    sayHello();
    sayHello("张三");
    
    // 引用传递
    function addOne(&$num) {
        $num++;
    }
    $x = 10;
    addOne($x);
    echo $x; // 11
    
    // 类型声明（PHP 7+）
    function calculate(int $a, float $b): float {
        return $a + $b;
    }
    
    // 返回多个值
    function getUser() {
        return [
            'name' => "张三",
            'age' => 25
        ];
    }
    [$name, $age] = getUser();
    
    // void返回类型（PHP 7.1+）
    function logMessage(string $msg): void {
        echo "[$msg]";
    }
?>
常见错误
  • 默认参数必须放在非默认参数之后
  • 引用传递时未使用&符号
  • 类型声明后传入错误类型
  • 尝试返回void函数的值
练习题
  1. 定义带默认参数的函数
  2. 练习引用传递
  3. 使用类型声明
  4. 返回多个值并解构

11 数组操作函数

▼
知识点
  • array_push/pop：末尾添加/删除
  • array_shift/unshift：开头添加/删除
  • array_merge：合并数组
  • array_search：搜索值
  • sort/rsort/asort/ksort：排序
代码示例
array-functions.php 
<?php
    $fruits = ["苹果", "香蕉"];
    
    // 添加元素
    array_push($fruits, "橙子");
    array_unshift($fruits, "葡萄");
    
    // 删除元素
    $last = array_pop($fruits);
    $first = array_shift($fruits);
    
    // 合并数组
    $arr1 = [1, 2];
    $arr2 = [3, 4];
    $merged = array_merge($arr1, $arr2);
    
    // 搜索
    $key = array_search("香蕉", $fruits);
    
    // 排序
    $nums = [3, 1, 4, 2];
    sort($nums); // 升序
    rsort($nums); // 降序
    
    // 关联数组排序
    $ages = ["张三" => 25, "李四" => 20];
    asort($ages); // 按值排序
    ksort($ages); // 按键排序
    
    // 数组切片
    $slice = array_slice($nums, 1, 2);
?>
常见错误
  • 对空数组使用pop/shift
  • 排序后索引被重置
  • array_search返回false时未正确判断
  • 合并关联数组时相同键名被覆盖
练习题
  1. 练习数组的增删操作
  2. 合并多个数组
  3. 对数组进行各种排序

13 日期与时间

▼
知识点
  • date()：格式化日期时间
  • time()：获取当前时间戳
  • strtotime()：字符串转时间戳
  • DateTime类：面向对象方式
  • 时区设置：date_default_timezone_set()
代码示例
datetime.php 
<?php
    // 设置时区
    date_default_timezone_set("Asia/Shanghai");
    
    // 获取当前时间戳
    $timestamp = time();
    
    // 格式化日期
    echo date("Y-m-d H:i:s"); // 2024-01-01 12:00:00
    echo date("l, F j, Y");
    
    // 字符串转时间戳
    $nextWeek = strtotime("+1 week");
    echo date("Y-m-d", $nextWeek);
    
    // DateTime类（推荐）
    $date = new DateTime();
    echo $date->format("Y-m-d H:i:s");
    
    // DateTime运算
    $date->add(new DateInterval("P1D"));
    echo $date->format("Y-m-d");
?>
常见错误
  • 忘记设置时区导致警告
  • strtotime解析失败返回false
  • date()函数参数顺序错误
  • DateTime比较时未使用getTimestamp()

    imagejpeg($dst, "thumbnail.jpg", 80);
    
    $white = imagecolorallocate($dst, 255, 255, 255);
    $font = "arial.ttf";
    imagettftext($dst, 12, 0, 10, 20, $white, $font, "水印文字");
    
    imagedestroy($src);
    imagedestroy($dst);
?>

  • 图片路径错误
  • 输出图片前有其他输出
  • 字体文件路径错误

Laravel路由和控制器 
<?php
    // routes/web.php
    use App\Http\Controllers\UserController;
    
    Route::get('/users', [UserController::class, 'index']);
    Route::get('/users/{id}', [UserController::class, 'show']);
    Route::post('/users', [UserController::class, 'store']);
    
    // app/Http/Controllers/UserController.php
    namespace App\Http\Controllers;
    
    use App\Models\User;
    
    class UserController extends Controller {
        public function index() {
            return User::all();
        }
        
        public function show($id) {
            return User::findOrFail($id);
        }
        
        public function store(Request $request) {
            $validated = $request->validate([
                'name' => 'required|string',
                'email' => 'required|email|unique:users'
            ]);
            return User::create($validated);
        }
    }
?>
