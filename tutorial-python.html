    1. Python简介与历史

    🌟 知识点讲解

    • Python的诞生：Python由Guido van Rossum于1989年底发明，1991年首次发布，名字来源于英国喜剧团体Monty Python
    • 设计哲学：强调代码可读性和简洁性，'优雅'、'明确'、'简单'是Python的设计核心
    • 版本演进：Python 2.x于2000年发布，Python 3.x于2008年发布，Python 2已于2020年停止维护
    • 应用领域：Web开发、数据科学、人工智能、自动化运维、科学计算、网络爬虫等
    • 语言特点：解释型、动态类型、强类型、面向对象、跨平台、丰富的标准库
    python
    # Python之禅 - 输入 import this 查看
import this

# 查看Python版本
import sys
print(sys.version)
print(sys.version_info)

# 第一个Python程序
print("Hello, World!")
print("欢迎使用Python教程！")

    ❌ 常见错误

    混淆Python 2和Python 3：print "hello" 是Python 2语法，Python 3必须使用 print("hello")

    中文编码问题：在Python 2中需要声明 # -*- coding: utf-8 -*-，Python 3默认UTF-8编码

    2. 安装与环境配置

    🌟 知识点讲解

    • 下载安装：从python.org下载对应操作系统的安装包，安装时务必勾选'Add Python to PATH'
    • 虚拟环境：使用venv创建隔离的Python环境，避免不同项目间的依赖冲突
    • 包管理器pip：pip是Python的包管理工具，用于安装和管理第三方库
    • IDE选择：推荐VS Code、PyCharm、Jupyter Notebook等开发工具
    python
    # 检查Python版本
python --version
python3 --version

# 创建虚拟环境
python -m venv myproject_env

# 激活虚拟环境（Windows）
myproject_env\Scripts\activate

# 激活虚拟环境（macOS/Linux）
source myproject_env/bin/activate

# 使用pip安装包
pip install requests
pip install numpy pandas matplotlib

# 查看已安装的包
pip list
pip freeze > requirements.txt

    ❌ 常见错误

    忘记勾选'Add Python to PATH'：导致命令行无法识别python命令，需要手动添加环境变量

    全局安装包：不使用虚拟环境直接安装包，可能导致版本冲突。应始终使用虚拟环境

    4. 注释与代码规范

    🌟 知识点讲解

    • 单行注释：使用#号开头，#后面的内容不会被Python执行
    • 多行注释：使用三引号（'''或"""）包裹，常用于函数/类的文档字符串
    • PEP 8规范：Python官方编码风格指南，规定缩进4空格、行宽79字符、命名规则等
    • 命名规范：变量/函数用snake_case，类用PascalCase，常量用UPPER_CASE
    python
    # 这是单行注释
name = "Python"  # 行末注释

"""
这是多行注释
可以写多行说明文字
通常用于文档字符串
"""

# PEP 8 命名规范
student_name = "Alice"       # 变量: snake_case
MAX_SIZE = 100               # 常量: UPPER_CASE

def calculate_area(radius):  # 函数: snake_case
    """计算圆的面积"""
    return 3.14 * radius ** 2

class StudentInfo:           # 类: PascalCase
    """学生信息类"""
    pass

    ❌ 常见错误

    注释未闭合：多行注释忘记闭合三引号，导致后面的代码全被当作注释

    中文字符问题：注释中使用中文时确保文件编码为UTF-8

    5. 变量与赋值

    🌟 知识点讲解

    • 变量定义：Python是动态类型语言，变量无需声明类型，直接赋值即可创建
    • 赋值方式：支持多重赋值（a=b=c=1）、解包赋值（a,b=1,2）、增强赋值（a+=1）
    • 命名规则：只能包含字母、数字、下划线，不能以数字开头，不能使用关键字
    • 变量作用：变量是数据的标签（引用），而非存储数据的容器
    python
    # 基本赋值
name = "Alice"
age = 25
height = 1.68

# 多重赋值
a = b = c = 100

# 解包赋值
x, y, z = 1, 2, 3

# 交换变量（Python特色）
x, y = y, x

# 增强赋值
count = 0
count += 1   # 等同于 count = count + 1
count -= 1
count *= 2

# 删除变量
del count

    ❌ 常见错误

    变量名以数字开头：1name = "test" 语法错误，变量名不能以数字开头

    使用关键字作变量名：class = "Math" 会报错，class是Python关键字

    6. 数字类型(int,float,complex)

    🌟 知识点讲解

    • int整数：支持任意大小的整数，没有溢出问题，支持二进制(0b)、八进制(0o)、十六进制(0x)
    • float浮点数：双精度64位，注意浮点精度问题（0.1+0.2≠0.3）
    • complex复数：形式为a+bj，实部和虚部都是浮点数
    • 数字运算：支持+、-、*、/、//（整除）、%（取模）、**（幂运算）
    python
    # 整数
a = 42
big_num = 999999999999999999999  # 任意大小
binary = 0b1010   # 二进制 = 10
hex_num = 0xA     # 十六进制 = 10

# 浮点数
pi = 3.14159
sci = 2.5e10      # 科学计数法

# 复数
c = 3 + 4j
print(c.real)     # 实部: 3.0
print(c.imag)     # 虚部: 4.0

# 运算
print(10 / 3)     # 3.3333... 普通除法
print(10 // 3)    # 3 整除
print(10 % 3)     # 1 取模
print(2 ** 10)    # 1024 幂运算

# 浮点精度问题
print(0.1 + 0.2)  # 0.30000000000000004
from decimal import Decimal
print(Decimal('0.1') + Decimal('0.2'))  # 0.3

    ❌ 常见错误

    整除与普通除法混淆：10/3 返回3.333，10//3 返回3，注意区分

    浮点精度问题：0.1 + 0.2 == 0.3 返回False，需要用decimal模块处理精确计算

    7. 字符串基础

    🌟 知识点讲解

    • 创建字符串：使用单引号、双引号或三引号创建，三引号可跨行
    • 索引与切片：支持正向索引(0起)和反向索引(-1起)，切片语法s[start:end:step]
    • 字符串不可变：创建后不能修改，修改操作会创建新字符串
    • 转义字符：\n换行、\t制表、\\反斜杠、\"双引号等
    python
    # 创建字符串
s1 = 'Hello'
s2 = "World"
s3 = """多行
字符串"""

print(text[0:3])   # Pyt
print(text[2:])    # thon

# 字符串拼接

# 转义字符
print("换行:\n制表:\t引号:\"")

    字符串修改：s = "hello"; s[0] = "H" 报错，字符串不可变，应使用 s = "H" + s[1:]

    8. 字符串高级操作

    🌟 知识点讲解

    • 格式化方法：f-string(Python 3.6+)、str.format()、%格式化三种方式
    • 常用方法：split()分割、join()连接、replace()替换、strip()去空白、find()查找
    • 判断方法：startswith()、endswith()、isdigit()、isalpha()、isalnum()等
    • 大小写转换：upper()、lower()、title()、capitalize()、swapcase()
    python
    # f-string格式化
name, age = "Alice", 25

    9. 布尔类型与None

    🌟 知识点讲解

    • 布尔值：True和False是bool类型的两个实例，首字母必须大写
    • Falsy值：False、0、0.0、""、[]、()、{}、set()、None在布尔上下文中均为False
    • None：表示'无'或'空'，是NoneType的唯一实例，常作为函数默认返回值
    • 判断相等：用is判断None（x is None），用==判断布尔值
    python
    # 布尔值
is_student = True
is_teacher = False

# bool()转换
print(bool(1))       # True
print(bool(0))       # False
print(bool(""))      # False
print(bool("hello")) # True
print(bool([]))      # False
print(bool([1]))     # True

# None
result = None
if result is None:
    print("结果为空")

# 布尔运算
print(True and False)  # False
print(True or False)   # True
print(not True)        # False

    ❌ 常见错误

    小写布尔值：true = 1 不会报错但创建了变量，正确的是 True

    用==判断None：if x == None 能工作但不够严谨，应使用 if x is None

    10. 类型转换

    🌟 知识点讲解

    • 隐式转换：Python自动将低精度类型转为高精度，如int+float自动转float
    • 显式转换：使用int()、float()、str()、bool()、list()等内置函数强制转换
    • 常见转换：字符串与数字互转、列表与元组互转、ASCII与字符互转
    • type()和isinstance()：用type()查看类型，isinstance()判断是否为某类型
    python
    # 显式转换
print(int(3.9))        # 3 截断小数
print(float(5))        # 5.0
print(str(100))        # "100"
print(bool(0))         # False
print(list("abc"))     # ['a', 'b', 'c']
print(tuple([1,2,3]))  # (1, 2, 3)

# 字符串转数字
print(int("42"))       # 42
print(float("3.14"))   # 3.14

# ASCII与字符
print(chr(65))         # A
print(ord('A'))        # 65

# 类型判断
print(type(42))        # 
print(isinstance(42, int))    # True

    无效转换：int("3.14") 报错，应先转float再转int：int(float("3.14"))

    非数字字符串：int("abc") 会抛出ValueError

    12. 比较与逻辑运算符

    🌟 知识点讲解

    • 比较运算符：==、!=、>、<、>=、<=，返回布尔值
    • 逻辑运算符：and（与）、or（或）、not（非），有短路求值特性
    • is与==区别：is比较对象身份（内存地址），==比较值
    • 链式比较：1 < x < 10 是合法的Python语法
    python
    # 比较运算符
print(5 == 5)      # True
print(5 != 3)      # True
print(5 > 3)       # True

# 逻辑运算符
print(True and False)  # False
print(True or False)   # True
print(not True)        # False

# 短路求值
result = 0 or "default"   # "default"

# is vs ==
a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)   # True  值相等
print(a is b)   # False 不同对象

# 链式比较
x = 5
print(1 < x < 10)  # True

    ❌ 常见错误

    混淆is和==：a is b 比较的是对象身份，通常应使用 a == b 比较值

    赋值与比较混淆：if x = 5 语法错误，应为 if x == 5

    True and False or True 的结果是什么？

    13. 赋值与位运算符

    🌟 知识点讲解

    • 赋值运算符：=、+=、-=、*=、/=、//=、%=、**=、&=、|=、^=、<<=、>>=
    • 位运算符：&（与）、|（或）、^（异或）、~（取反）、<<（左移）、>>（右移）
    • 海象运算符：:=（Python 3.8+）在表达式中赋值
    • 位运算应用：权限控制、标志位、快速计算2的幂
    python
    # 位运算
print(5 & 3)    # 1   101 & 011 = 001
print(5 | 3)    # 7   101 | 011 = 111
print(5 ^ 3)    # 6   101 ^ 011 = 110
print(~5)       # -6  取反
print(5 << 1)   # 10  左移1位相当于*2
print(5 >> 1)   # 2   右移1位相当于//2

# 位运算实用技巧
def is_even(n):
    return (n & 1) == 0

print(is_even(4))  # True
print(is_even(7))  # False

    ❌ 常见错误

    混淆逻辑运算与位运算：& 是位与，and 是逻辑与，在条件判断中应使用and

    负数位移：-1 >> 1 结果是-1，不是0

    14. if条件语句

    🌟 知识点讲解

    • 基本语法：if 条件: 后跟缩进的代码块，条件为True时执行
    • 条件表达式：任何能转换为布尔值的表达式，非零/非空即为True
    • 嵌套if：if语句内部可以再嵌套if，注意缩进层级
    • 代码块：用缩进（4空格）标识，同一缩进级别为同一代码块

    if vip:

    15. elif与else

    🌟 知识点讲解

    • elif：'else if'的缩写，用于多条件分支，从上到下依次判断
    • else：所有条件都不满足时执行的兜底分支
    • 匹配模式(Python 3.10+)：match-case语句类似switch-case
    • 只执行一个分支：条件从上到下判断，匹配第一个后跳出整个if-elif-else
    python
    # 多条件判断
score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"
print(f"等级: {grade}")  # B

    ❌ 常见错误

    条件重叠：if x > 5: ... elif x > 3: ... 当x=6时只执行第一个分支

    遗漏else：忘记处理所有条件都不满足的情况

    📝 练习题

    month = 4 if 3 <= month <= 5: print("春季") elif 6 <= month <= 8: print("夏季") elif 9 <= month <= 11: print("秋季") else: print("冬季")

    16. 三元表达式

    🌟 知识点讲解

    • 语法：值1 if 条件 else 值2，条件为True返回值1，否则返回值2
    • 简洁替代：用于简化简单的if-else，使代码更紧凑
    • 嵌套使用：可以嵌套但可读性降低，不建议超过两层
    python
    # 三元表达式
age = 20
status = "成年" if age >= 18 else "未成年"
print(status)  # 成年

# 实用场景
x, y = 5, 10
max_val = x if x > y else y   # 10

# 嵌套（不推荐过于复杂）
score = 75

    19. break与continue

    🌟 知识点讲解

    • 写入方法：write()写入字符串、writelines()写入列表
    python
    # 读取文件
with open("example.txt", "r") as f:
    content = f.read()
    print(content)

# 写入文件
with open("output.txt", "w") as f:
    f.write("Hello, World!")

# 追加内容
with open("output.txt", "a") as f:
    f.write("\nThis is appended.")

# 按行读取
with open("example.txt", "r") as f:
    for line in f:
        print(line.strip())

    35. 异常处理

    🌟 知识点讲解

    • try-except语句：try中放可能出错的代码，except中处理异常
    • 多个except：可以捕获不同类型的异常，except Exception as e捕获所有异常
    • else和finally：else在无异常时执行，finally无论是否异常都执行
    • raise语句：主动抛出异常，可自定义异常类
    python
    # 基本错误处理
try:
    num = int("abc")
except ValueError:
    print("请输入有效的数字")

# 多个except
try:
    result = 10 / 0
except ZeroDivisionError:
    print("不能除以零")
except Exception as e:
    print(f"错误: {e}")

# else和finally
try:
    f = open("file.txt")
except FileNotFoundError:
    print("文件不存在")
else:
    print("文件读取成功")
finally:
    print("操作完成")

# 自定义异常
class InvalidAgeError(Exception):
    pass

def check_age(age):
    if age < 0:
        raise InvalidAgeError("年龄不能为负数")

    ❌ 常见错误

    过度捕获异常：使用 except Exception 会捕获所有异常，应尽量捕获特定异常

    空的except块会隐藏错误，不利于调试

    📝 练习题

    编写代码处理除以零的异常
    try: result = 10 / 0 except ZeroDivisionError: print("错误: 不能除以零")
