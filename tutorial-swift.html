    1. Swift简介

    🌟 知识点讲解

    • 诞生背景：Swift由Apple于2014年WWDC发布，由Chris Lattner主导设计，旨在替代Objective-C成为Apple平台主力开发语言
    • 设计理念：安全、快速、表达力强，融合了Objective-C、Rust、Haskell等语言的优点
    • 开源之路：2015年12月开源，支持Linux平台，社区活跃，版本迭代快速
    • 应用场景：iOS/macOS/watchOS/tvOS应用开发、服务端开发(Vapor/Perfect)、机器学习(Swift for TensorFlow)
    • 核心特性：类型安全、可选类型、闭包、协议导向编程、值类型优先、自动引用计数(ARC)
    swift
    // Swift版本信息
// Swift 5.9+ 是当前主流版本

// Swift是类型安全的语言

let year: Int = 2014
print("\(language)诞生于\(year)年")

    ❌ 常见错误

    混淆Swift与SwiftUI：Swift是编程语言，SwiftUI是UI框架，两者不是同一个概念

    以为Swift只能在Apple平台使用：Swift已开源，也支持Linux和Windows上的服务端开发

    3. 第一个Swift程序

    🌟 知识点讲解

    • print函数：使用print()输出内容到控制台，支持字符串插值 \(变量名)
    • 程序入口：Swift程序从源文件顶部代码开始执行，不需要main函数（Swift 5.3+），也可用@main标记入口
    • 字符串插值：使用\(expression)在字符串中嵌入表达式，这是Swift的特色功能
    • 语句结尾：Swift语句末尾不需要分号，但同一行多条语句需要分号分隔
    swift
    // 第一个Swift程序
print("Hello, World!")

// 字符串插值

let b = 20

// 多条语句同行需要分号
let x = 1; let y = 2
print("x=\(x), y=\(y)")

    4. 注释与分号

    🌟 知识点讲解

    • 单行注释：使用 // 开头，// 后面的内容不会被执行
    • 多行注释：使用 /* */ 包裹，支持嵌套注释（Swift特色，不同于C/Java）
    • 文档注释：使用 /// 开头生成Quick Help文档，Xcode可自动显示
    • 分号规则：Swift语句末尾无需分号，但同一行多条语句必须用分号分隔
    swift
    // 这是单行注释
let greeting = "Hello"  // 行末注释

/* 这是多行注释
   可以跨越多行 */

/* Swift支持嵌套注释 /* 像这样 */
   在C语言中这是不允许的 */

/// 这是文档注释
/// 用于生成代码文档
/// - Parameter name: 用户名
/// - Returns: 问候语
func sayHello(to name: String) -> String {
    return "Hello, \(name)!"
}

// 同一行多条语句需要分号
let a = 1; let b = 2; print(a + b)

    ❌ 常见错误

    多行注释未闭合：忘记写 */ 导致后续代码都被当作注释

    不必要的分号：Swift中每行末尾加分号虽然不报错，但不符合代码规范，应省略

    5. 变量与常量

    🌟 知识点讲解

    • let常量：使用let声明常量，一旦赋值不可修改，Swift推荐优先使用let
    • var变量：使用var声明变量，值可以修改，仅在需要改变值时使用var
    • 类型推断：Swift编译器能自动推断变量类型，无需显式声明，但也可显式指定类型
    • 命名规则：支持Unicode字符命名，不能包含数学符号、箭头、空白字符，不能以数字开头

    // 常量声明

    6. 整数与浮点数

    🌟 知识点讲解

    • 整数类型：Int（有符号，默认）、UInt（无符号），以及Int8/16/32/64等固定位宽类型
    • 浮点类型：Double（64位，默认浮点推断类型）、Float（32位），精度分别为15位和6位

    • 数值可读性：可用下划线分隔大数字提升可读性，如 1_000_000

    7. 布尔与元组

    🌟 知识点讲解

    • Bool类型：只有true和false两个值，Swift不支持非零即真，条件判断必须是Bool类型
    • 元组(Tuple)：将多个值组合为单一复合值，使用括号创建，可包含不同类型
    • 元组解构：可将元组的元素分解为单独的常量或变量，也可通过索引或命名访问
    • 元组适用场景：函数返回多个值、临时组合数据，不适合复杂数据结构（应用结构体）
    swift
    // 布尔类型

let isOld = false

    8. 可选类型(Optional)

    🌟 知识点讲解

    • 可选类型概念：Optional表示变量可能有值也可能为nil，是Swift安全的核心特性
    • 声明可选：在类型后加问号，如 var name: String? 表示name可能为nil
    • 强制解包：使用!强制获取可选值，若为nil则运行时崩溃，务必确定非nil才使用
    • 可选绑定：使用if let或guard let安全解包，是推荐的可选值处理方式
    • 隐式解包：使用!声明(如var x: Int!)，访问时自动解包，但为nil仍会崩溃
    swift
    // 声明可选类型
var name: String? = "Alice"
var address: String? = nil

// 强制解包（危险！）
print(name!)  // Alice
// print(address!)  // 运行时崩溃！

// 可选绑定（安全）
if let userName = name {
    print("用户名: \(userName)")
} else {
    print("用户名为空")
}

// guard let（提前退出）
func greet(_ name: String?) {
    guard let name = name else {
        print("没有名字")
        return
    }
    print("你好, \(name)")
}

// 可选链
let upperName = name?.uppercased()  // 可能为nil

// nil合并运算符 ??
let displayName = name ?? "匿名用户"

// 隐式解包可选
var forcedName: String! = "Bob"
print(forcedName)  // 直接使用，不需要!

    ❌ 常见错误

    强制解包nil：var x: Int? = nil; print(x!) 运行时崩溃，应使用可选绑定

    忘记解包：let len = name?.count 结果仍为Int?，不能直接当Int使用

    10. 字符串操作

    🌟 知识点讲解

    • 拼接与修改：使用+和+=拼接字符串，append()追加字符，String是值类型修改会创建新副本
    • 索引操作：使用startIndex、endIndex和index(_:offsetBy:)访问指定位置字符
    • 子字符串：使用下标获取子串，结果为SubString类型（共享原字符串内存），用完应转为String
    • 常用方法：uppercased()、lowercased()、contains()、replacingOccurrences()、split()等
    swift
    // 拼接
var str = "Hello"
str += ", Swift"    // "Hello, Swift"
str.append("!")     // "Hello, Swift!"

// 索引操作
let text = "Swift"
let first = text[text.startIndex]  // "S"
let last = text[text.index(before: text.endIndex)]  // "t"
let third = text[text.index(text.startIndex, offsetBy: 2)]  // "i"

// 插入与删除
var greeting = "Helo"
greeting.insert("l", at: greeting.index(greeting.startIndex, offsetBy: 2))
greeting.remove(at: greeting.index(before: greeting.endIndex))

// 子字符串
let full = "Hello, World!"
let index = full.firstIndex(of: ",")!
let sub = full[full.startIndex..

    ❌ 常见错误

    使用整数索引：text[0] 在Swift中编译错误，必须使用text[text.startIndex]

    SubString未转String：SubString共享原字符串内存，长期持有会阻止原字符串释放，用完应转String

    11. 类型转换

    🌟 知识点讲解

    • 数值类型转换：Int与Double/Float之间必须显式转换，Swift不会隐式转换
    • 整数与浮点互转：Int(3.14)截断为3，Double(3)变为3.0
    • 字符串与数值转换：使用Int("42")等初始化器，返回可选值因为可能转换失败
    • 类型判断：使用is运算符判断类型，使用as和as?进行类型转换
    swift
    // 数值类型转换
let intVal: Int = 42
let doubleVal: Double = Double(intVal)  // 42.0
let floatVal: Float = Float(intVal)     // 42.0
let backToInt: Int = Int(3.99)          // 3（截断）

// 混合运算必须统一类型
let a: Int = 10
let b: Double = 3.14
// let c = a + b  // 编译错误！
let c = Double(a) + b  // 13.14

// 字符串与数值转换
let strToNum = Int("42")         // Optional(42)
let strToDouble = Double("3.14") // Optional(3.14)
let invalid = Int("hello")       // nil
let numToStr = String(42)        // "42"

// 安全转换
if let number = Int("123") {
    print("转换成功: \(number)")
} else {
    print("转换失败")
}

// 类型检查与转换
class Animal {}
class Dog: Animal {}
let pet: Animal = Dog()
if pet is Dog {
    let dog = pet as! Dog
    print("这是一只狗")
}
// 安全转换
if let dog = pet as? Dog {
    print("安全转换: 这是一只狗")
}

    ❌ 常见错误

    隐式类型转换：let x: Double = 42 编译错误，需写 let x: Double = 42.0 或 Double(42)

    忽略转换失败：Int("abc")! 强制解包会崩溃，应用if let安全解包

    12. 运算符

    🌟 知识点讲解

    • 算术运算符：+、-、*、/、%（取余），注意整数除法截断，溢出运算符&+、&-、&*
    • 比较运算符：==、!=、>、<、>=、<=，以及===和!==（引用比较）

    // 算术运算
print(10 + 3)    // 13
print(10 - 3)    // 7
print(10 * 3)    // 30
print(10 / 3)    // 3（整数除法）
print(10 % 3)    // 1

// 比较运算

    13. if-else语句

    🌟 知识点讲解

    • 基本语法：if 条件 { } else { }，条件必须是Bool类型，大括号不可省略
    • 多条件：使用else if处理多个条件分支，从上到下依次判断
    • 条件绑定：在if条件中使用if let解包可选值，同时判断是否为nil
    • guard语句：提前退出，当条件不满足时立即return，避免嵌套过深
    swift
    // 基本if-else
let age = 20
if age >= 18 {
    print("成年人")
} else {
    print("未成年人")
}

// 多条件判断
let score = 85
if score >= 90 {
    print("优秀")
} else if score >= 80 {
    print("良好")
} else if score >= 60 {
    print("及格")
} else {
    print("不及格")
}

// 条件绑定
let userName: String? = "Alice"
if let name = userName {
    print("欢迎, \(name)")
} else {
    print("请登录")
}

// guard语句
func greet(_ name: String?) {
    guard let name = name else {
        print("没有名字")
        return
    }
    print("你好, \(name)")
}

// 组合条件
let isVIP = true
let balance = 100
if isVIP && balance >= 50 {
    print("VIP优惠")
}

    ❌ 常见错误

    省略大括号：if condition { } 的大括号不可省略，即使只有一行代码

    条件非Bool：if score { } 编译错误，必须写 if score > 0 { }

    20. 闭包

    🌟 知识点讲解

    }

    31. 内存管理
