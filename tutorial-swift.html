    1. Swift简介

    🌟 知识点讲解

    • 诞生背景：Swift由Apple于2014年WWDC发布，由Chris Lattner主导设计，旨在替代Objective-C成为Apple平台主力开发语言
    • 设计理念：安全、快速、表达力强，融合了Objective-C、Rust、Haskell等语言的优点
    • 开源之路：2015年12月开源，支持Linux平台，社区活跃，版本迭代快速
    • 应用场景：iOS/macOS/watchOS/tvOS应用开发、服务端开发(Vapor/Perfect)、机器学习(Swift for TensorFlow)
    • 核心特性：类型安全、可选类型、闭包、协议导向编程、值类型优先、自动引用计数(ARC)
    swift
    // Swift版本信息
// Swift 5.9+ 是当前主流版本

// Swift是类型安全的语言

let year: Int = 2014
print("\(language)诞生于\(year)年")

    ❌ 常见错误

    混淆Swift与SwiftUI：Swift是编程语言，SwiftUI是UI框架，两者不是同一个概念

    以为Swift只能在Apple平台使用：Swift已开源，也支持Linux和Windows上的服务端开发

    3. 第一个Swift程序

    🌟 知识点讲解

    • print函数：使用print()输出内容到控制台，支持字符串插值 \(变量名)
    • 程序入口：Swift程序从源文件顶部代码开始执行，不需要main函数（Swift 5.3+），也可用@main标记入口
    • 字符串插值：使用\(expression)在字符串中嵌入表达式，这是Swift的特色功能
    • 语句结尾：Swift语句末尾不需要分号，但同一行多条语句需要分号分隔

    // 第一个Swift程序

    4. 注释与分号

    🌟 知识点讲解

    • 单行注释：使用 // 开头，// 后面的内容不会被执行
    • 多行注释：使用 /* */ 包裹，支持嵌套注释（Swift特色，不同于C/Java）
    • 文档注释：使用 /// 开头生成Quick Help文档，Xcode可自动显示
    • 分号规则：Swift语句末尾无需分号，但同一行多条语句必须用分号分隔
    swift
    // 这是单行注释
let greeting = "Hello"  // 行末注释

/* 这是多行注释
   可以跨越多行 */

/* Swift支持嵌套注释 /* 像这样 */
   在C语言中这是不允许的 */

/// 这是文档注释
/// 用于生成代码文档
/// - Parameter name: 用户名
/// - Returns: 问候语
func sayHello(to name: String) -> String {
    return "Hello, \(name)!"

let a = 1; let b = 2; print(a + b)

    多行注释未闭合：忘记写 */ 导致后续代码都被当作注释

    不必要的分号：Swift中每行末尾加分号虽然不报错，但不符合代码规范，应省略

    5. 变量与常量

    🌟 知识点讲解

    • let常量：使用let声明常量，一旦赋值不可修改，Swift推荐优先使用let
    • var变量：使用var声明变量，值可以修改，仅在需要改变值时使用var

    6. 整数与浮点数

    🌟 知识点讲解

    7. 布尔与元组

    🌟 知识点讲解

    • Bool类型：只有true和false两个值，Swift不支持非零即真，条件判断必须是Bool类型
    • 元组(Tuple)：将多个值组合为单一复合值，使用括号创建，可包含不同类型

    • 元组适用场景：函数返回多个值、临时组合数据，不适合复杂数据结构（应用结构体）

    // 布尔类型

    8. 可选类型(Optional)

    🌟 知识点讲解

    • 可选类型概念：Optional表示变量可能有值也可能为nil，是Swift安全的核心特性
    • 声明可选：在类型后加问号，如 var name: String? 表示name可能为nil
    • 强制解包：使用!强制获取可选值，若为nil则运行时崩溃，务必确定非nil才使用
    • 可选绑定：使用if let或guard let安全解包，是推荐的可选值处理方式
    • 隐式解包：使用!声明(如var x: Int!)，访问时自动解包，但为nil仍会崩溃
    swift
    // 声明可选类型
var name: String? = "Alice"
var address: String? = nil

// 强制解包（危险！）
print(name!)  // Alice
// print(address!)  // 运行时崩溃！

// 可选绑定（安全）
if let userName = name {
    print("用户名: \(userName)")
} else {
    print("用户名为空")
}

// guard let（提前退出）
func greet(_ name: String?) {
    guard let name = name else {
        print("没有名字")
        return
    }
    print("你好, \(name)")
}

// 可选链
let upperName = name?.uppercased()  // 可能为nil

// nil合并运算符 ??
let displayName = name ?? "匿名用户"

// 隐式解包可选
var forcedName: String! = "Bob"
print(forcedName)  // 直接使用，不需要!

    ❌ 常见错误

    强制解包nil：var x: Int? = nil; print(x!) 运行时崩溃，应使用可选绑定

    忘记解包：let len = name?.count 结果仍为Int?，不能直接当Int使用

    10. 字符串操作

    🌟 知识点讲解

    • 拼接与修改：使用+和+=拼接字符串，append()追加字符，String是值类型修改会创建新副本
    • 索引操作：使用startIndex、endIndex和index(_:offsetBy:)访问指定位置字符
    • 子字符串：使用下标获取子串，结果为SubString类型（共享原字符串内存），用完应转为String
    • 常用方法：uppercased()、lowercased()、contains()、replacingOccurrences()、split()等
    swift
    // 拼接
var str = "Hello"
str += ", Swift"    // "Hello, Swift"
str.append("!")     // "Hello, Swift!"

// 索引操作
let text = "Swift"
let first = text[text.startIndex]  // "S"
let last = text[text.index(before: text.endIndex)]  // "t"
let third = text[text.index(text.startIndex, offsetBy: 2)]  // "i"

// 插入与删除
var greeting = "Helo"
greeting.insert("l", at: greeting.index(greeting.startIndex, offsetBy: 2))
greeting.remove(at: greeting.index(before: greeting.endIndex))

// 子字符串
let full = "Hello, World!"
let index = full.firstIndex(of: ",")!
let sub = full[full.startIndex..

    ❌ 常见错误

    使用整数索引：text[0] 在Swift中编译错误，必须使用text[text.startIndex]

    SubString未转String：SubString共享原字符串内存，长期持有会阻止原字符串释放，用完应转String

    11. 类型转换

    🌟 知识点讲解

    • 数值类型转换：Int与Double/Float之间必须显式转换，Swift不会隐式转换
    • 整数与浮点互转：Int(3.14)截断为3，Double(3)变为3.0
    • 字符串与数值转换：使用Int("42")等初始化器，返回可选值因为可能转换失败
    • 类型判断：使用is运算符判断类型，使用as和as?进行类型转换
    swift
    // 数值类型转换
let intVal: Int = 42
let doubleVal: Double = Double(intVal)  // 42.0
let floatVal: Float = Float(intVal)     // 42.0
let backToInt: Int = Int(3.99)          // 3（截断）

// 混合运算必须统一类型
let a: Int = 10
let b: Double = 3.14
// let c = a + b  // 编译错误！
let c = Double(a) + b  // 13.14

// 字符串与数值转换
let strToNum = Int("42")         // Optional(42)
let strToDouble = Double("3.14") // Optional(3.14)
let invalid = Int("hello")       // nil
let numToStr = String(42)        // "42"

// 安全转换
if let number = Int("123") {
    print("转换成功: \(number)")
} else {
    print("转换失败")
}

// 类型检查与转换
class Animal {}
class Dog: Animal {}
let pet: Animal = Dog()
if pet is Dog {
    let dog = pet as! Dog
    print("这是一只狗")
}
// 安全转换
if let dog = pet as? Dog {
    print("安全转换: 这是一只狗")
}

    ❌ 常见错误

    隐式类型转换：let x: Double = 42 编译错误，需写 let x: Double = 42.0 或 Double(42)

    忽略转换失败：Int("abc")! 强制解包会崩溃，应用if let安全解包

    12. 运算符

    🌟 知识点讲解

    • 算术运算符：+、-、*、/、%（取余），注意整数除法截断，溢出运算符&+、&-、&*
    • 比较运算符：==、!=、>、<、>=、<=，以及===和!==（引用比较）
    • 逻辑运算符：!（非）、&&（与）、||（或），短路求值

    • nil合并：?? 运算符，为nil时提供默认值
    swift
    // 算术运算
print(10 + 3)    // 13
print(10 - 3)    // 7
print(10 * 3)    // 30
print(10 / 3)    // 3（整数除法）
print(10 % 3)    // 1

// 溢出运算符
let overflow = UInt8.max &+ 1  // 溢出后回到0

// 比较运算
print(5 > 3)     // true
print("apple" < "banana")  // true（字典序）

// 逻辑运算
let x = true, y = false
print(!x)         // false

for i in 1...5 { print(i) }    // 1,2,3,4,5
for i in 1..<5 { print(i) }    // 1,2,3,4

// nil合并
let nickname: String? = nil

    范围运算符方向：3...1 是空范围，遍历不会执行，范围必须a ≤ b

    13. if-else语句

    🌟 知识点讲解

    • 基本语法：if条件{ }，条件不需要括号（可加），大括号不可省略
    • 多分支：if-else if-else结构，从上到下匹配，执行第一个满足条件的分支
    • 条件必须是Bool：Swift严格要求条件为Bool类型，不能用整数代替布尔值
    • 可选绑定：if let x = optionalValue { } 同时判断和解包可选值
    swift
    // 基本if-else
let score = 85
if score >= 90 {
    print("优秀")
} else if score >= 80 {
    print("良好")
} else if score >= 60 {
    print("及格")
} else {
    print("不及格")
}

// 条件必须是Bool
let count = 5
// if count { }  // 编译错误！
if count > 0 {   // 正确
    print("count不为零")
}

// 可选绑定
let userInput: String? = "Hello"
if let input = userInput {
    print("用户输入: \(input)")
} else {
    print("没有输入")
}

// 多个可选绑定
let a: Int? = 10
let b: Int? = 20
if let x = a, let y = b {
    print("和: \(x + y)")
}

    ❌ 常见错误

    省略大括号：if score > 60 print("及格") 编译错误，Swift必须有大括号

    使用赋值作条件：if x = 10 { } 编译错误，Swift中=是赋值不是比较

    14. switch语句

    🌟 知识点讲解

    • 基本语法：switch匹配值，case分支，不需要break（Swift自动不贯穿），必须穷举所有情况
    • 模式匹配：支持范围匹配、元组匹配、值绑定、where条件过滤
    • 贯穿：使用fallthrough关键字显式贯穿到下一个case
    • 穷举要求：必须处理所有可能的情况，可用default兜底
    swift
    // 基本switch
let grade = "A"
switch grade {
case "A":
    print("优秀")
case "B":
    print("良好")
case "C":
    print("及格")
default:
    print("不及格")
}

// 范围匹配
let score = 85
switch score {
case 90...100:
    print("优秀")
case 80..<90:
    print("良好")
case 60..<80:
    print("及格")
default:
    print("不及格")
}

// 元组匹配与值绑定
let point = (1, 0)
switch point {
case (0, 0):
    print("原点")
case (_, 0):
    print("x轴上")
case (0, _):
    print("y轴上")
case let (x, y) where x == y:
    print("在对角线上")
case let (x, y):
    print("坐标: (\(x), \(y))")
}

// 复合case
let char: Character = "a"
switch char {
case "a", "e", "i", "o", "u":
    print("元音")
default:
    print("辅音")
}

    ❌ 常见错误

    缺少default：非穷举的switch必须包含default分支，否则编译错误

    期望贯穿：Swift的case默认不贯穿，若需要执行下一个case需用fallthrough

    📝 练习题

    用switch判断月份属于哪个季节
    let month = 4 switch month { case 3, 4, 5: print("春季") case 6, 7, 8: print("夏季") case 9, 10, 11: print("秋季") case 12, 1, 2: print("冬季") default: break }

    15. for-in循环

    🌟 知识点讲解

    • 遍历集合：for item in collection 遍历数组、字典、范围、字符串等序列
    • 范围遍历：for i in 1...5 遍历整数范围，使用..< 创建半开范围

    16. while与repeat-while

    🌟 知识点讲解

    • while循环：先判断条件再执行循环体，条件为false时不执行
    • repeat-while：先执行循环体再判断条件，至少执行一次（类似do-while）
    • 适用场景：循环次数未知时使用while，至少执行一次时使用repeat-while
    • 死循环：while true { } 创建无限循环，需在内部使用break退出
    swift
    // while循环
var count = 0
while count < 5 {
    print(count)
    count += 1
}

// repeat-while（至少执行一次）
var number = 0
repeat {
    print(number)
    number += 1
} while number < 3

// 实际应用：猜数字游戏
let target = 7
var guess = 0
repeat {
    guess += 1
    print("第\(guess)次猜")
} while guess != target

// while处理可选值
var items: [Int]? = [1, 2, 3]
while let list = items, !list.isEmpty {
    print("还有\(list.count)个元素")
    items = nil
}

    ❌ 常见错误

    忘记更新循环变量：while count < 5 { print(count) } 永远循环，需count += 1

    混淆while和repeat-while：while可能零次执行，repeat-while至少执行一次

    📝 练习题

    用while循环计算2的n次方首次超过1000时的n值

    17. 控制转移(break/continue)

    🌟 知识点讲解

    • break：立即终止整个循环，跳出循环体执行后续代码
    • continue：跳过当前迭代的剩余代码，进入下一次迭代
    • 标签语句：使用label:给循环起名，break/continue可指定跳出到哪层循环
    • return：从函数中返回，终止函数执行
    swift
    // break - 找到目标就停止
let numbers = [1, 3, 5, 7, 9, 2, 4]
for num in numbers {
    if num % 2 == 0 {
        print("找到偶数: \(num)")
        break
    }
}

// continue - 跳过奇数
for i in 1...10 {
    if i % 2 != 0 {
        continue
    }
    print(i)  // 只打印偶数
}

// 标签语句 - 嵌套循环中跳出外层
outerLoop: for i in 1...3 {
    for j in 1...3 {
        if i == 2 && j == 2 {
            print("在(2,2)处跳出外层循环")
            break outerLoop
        }
        print("(\(i), \(j))")
    }
}

// switch中的break
let value = 5
switch value {
case 1...10:
    break  // 不做任何事
default:
    print("其他")
}

    ❌ 常见错误

    switch空case：case语句不能为空，什么都不做需要写break

    嵌套循环中break只跳出内层：不加标签时break只影响最内层循环

    19. 集合Set

    🌟 知识点讲解

    • 集合特性：无序、唯一，元素必须可哈希(Hashable)，适合去重和集合运算
    • 创建集合：Set<Int>类型声明，或用字面量初始化，注意与数组区分
    • 集合运算：intersection(交集)、union(并集)、subtracting(差集)、symmetricDifference(对称差)
    • 关系判断：isSubset、isSuperset、isDisjoint判断集合间关系
    swift
    // 创建集合
var colors: Set = ["红", "绿", "蓝"]
var nums: Set = [1, 2, 3, 3, 2]  // {1, 2, 3}，自动去重

// 添加和删除
colors.insert("黄")
colors.remove("绿")

// 集合运算
let a: Set = [1, 2, 3, 4]
let b: Set = [3, 4, 5, 6]
print(a.intersection(b))          // {3, 4} 交集
print(a.union(b))                 // {1,2,3,4,5,6} 并集
print(a.subtracting(b))           // {1, 2} 差集
print(a.symmetricDifference(b))   // {1, 2, 5, 6} 对称差

// 关系判断
let small: Set = [1, 2]
let big: Set = [1, 2, 3, 4]
print(small.isSubset(of: big))    // true
print(big.isSuperset(of: small))  // true
print(small.isDisjoint(with: [5, 6]))  // true

// 去重应用
let withDup = [1, 1, 2, 2, 3, 3]
let unique = Array(Set(withDup))  // [1, 2, 3]（顺序不确定）

    ❌ 常见错误

    数组字面量误识别为Set：let s = [1, 2, 3] 默认创建Array，需显式声明类型Set

    依赖集合顺序：集合无序，每次遍历顺序可能不同，需要有序时应使用数组

    21. 函数定义

    22. 函数参数与返回值

    🌟 知识点讲解

    • 默认参数值：在参数声明时指定默认值，调用时可省略该参数
    • 可变参数：使用...声明(如Int...)，函数内当数组使用，一个函数最多一个可变参数
    • 输入输出参数：使用inout声明，可修改外部变量，调用时需加&
    • 可选元组返回：函数可返回可选元组，用于可能返回nil的场景
    swift
    // 默认参数值
func power(base: Int, exponent: Int = 2) -> Int {
    var result = 1
    for _ in 1...exponent { result *= base }
    return result
}
print(power(base: 3))          // 9（exponent默认2）
print(power(base: 3, exponent: 3))  // 27

// 可变参数
func average(_ numbers: Double...) -> Double {
    if numbers.isEmpty { return 0 }
    return numbers.reduce(0, +) / Double(numbers.count)
}
print(average(1, 2, 3, 4, 5))  // 3.0

// 输入输出参数
func swapValues(_ a: inout Int, _ b: inout Int) {
    let temp = a
    a = b
    b = temp
}
var x = 10, y = 20
swapValues(&x, &y)
print("x=\(x), y=\(y)")  // x=20, y=10

// 可选元组返回
func findMinMax(_ array: [Int]) -> (min: Int, max: Int)? {
    if array.isEmpty { return nil }
    return (array.min()!, array.max()!)
}
if let result = findMinMax([5, 3, 8, 1]) {
    print("min=\(result.min), max=\(result.max)")
}

    23. 函数类型

    🌟 知识点讲解

    • 函数类型：每个函数都有特定类型，由参数类型和返回类型组成，如 (Int, Int) -> Int
    • 函数作变量：可将函数赋值给变量或常量，像普通值一样传递
    • 函数作参数：可将函数作为另一个函数的参数，实现回调和高阶函数
    • 函数作返回值：函数可以返回另一个函数，实现函数工厂模式
    swift
    // 函数类型
func add(_ a: Int, _ b: Int) -> Int { return a + b }
func multiply(_ a: Int, _ b: Int) -> Int { return a * b }

// 函数作变量
var operation: (Int, Int) -> Int = add
print(operation(3, 4))    // 7
operation = multiply
print(operation(3, 4))    // 12

// 函数作参数
func calculate(_ a: Int, _ b: Int, using op: (Int, Int) -> Int) -> Int {
    return op(a, b)
}
print(calculate(10, 5, using: add))       // 15
print(calculate(10, 5, using: multiply))  // 50

// 函数作返回值
func stepForward(_ input: Int) -> Int { return input + 1 }
func stepBackward(_ input: Int) -> Int { return input - 1 }
func chooseStep(backward: Bool) -> (Int) -> Int {
    return backward ? stepBackward : stepForward
}
let step = chooseStep(backward: false)
print(step(5))  // 6

    函数类型不匹配：将(Int)->Int赋值给(Int,Int)->Int类型变量，编译错误

    24. 嵌套函数

    🌟 知识点讲解

    • 定义嵌套：在函数内部定义另一个函数，内部函数对外部不可见
    • 捕获值：嵌套函数可捕获外部函数的变量和常量，形成闭包
    • 封装逻辑：将辅助函数嵌套在主函数中，避免污染命名空间
    • 返回嵌套函数：外部函数可返回内部嵌套函数，实现有状态的函数
    swift
    // 基本嵌套函数
func calculateStatistics(scores: [Int]) -> (min: Int, max: Int, avg: Int) {
    func findMin() -> Int { scores.min()! }
    func findMax() -> Int { scores.max()! }
    func calcAvg() -> Int { scores.reduce(0, +) / scores.count }
    
    return (findMin(), findMax(), calcAvg())
}

let stats = calculateStatistics(scores: [85, 92, 78, 95, 88])
print("最低: \(stats.min), 最高: \(stats.max), 平均: \(stats.avg)")

// 返回嵌套函数
func makeIncrementer(by amount: Int) -> () -> Int {
    var total = 0
    func incrementer() -> Int {
        total += amount
        return total
    }
    return incrementer
}

let incBy5 = makeIncrementer(by: 5)
print(incBy5())  // 5
print(incBy5())  // 10
print(incBy5())  // 15

// 另一个独立计数器
let incBy10 = makeIncrementer(by: 10)
print(incBy10())  // 10

    ❌ 常见错误

    从外部访问嵌套函数：嵌套函数只在其定义的作用域内可见，不能从外部直接调用

    忘记返回嵌套函数：定义了嵌套函数但忘记返回它，外部无法使用

    25. 闭包基础

    🌟 知识点讲解

    • 闭包概念：自包含的函数代码块，可捕获和存储上下文中的常量和变量
    • 闭包表达式：{ (parameters) -> returnType in statements }，用in分隔声明和实现
    • 简写形式：Swift提供多种简写：省略类型、省略return、$0/$1参数缩写
    • 值捕获：闭包会捕获所在上下文的常量和变量，即使原作用域已不存在
    swift
    // 闭包表达式
let add: (Int, Int) -> Int = { (a: Int, b: Int) -> Int in
    return a + b
}
print(add(3, 5))  // 8

// 简写形式
let multiply: (Int, Int) -> Int = { $0 * $1 }
print(multiply(4, 6))  // 24

// 闭包作为函数参数
let names = ["Chris", "Alex", "Barry", "Daniella"]
let sorted1 = names.sorted(by: { (a: String, b: String) -> Bool in
    return a < b
})
let sorted2 = names.sorted { $0 < $1 }  // 最简形式
print(sorted2)

// 值捕获
func makeCounter() -> () -> Int {
    var count = 0
    return {
        count += 1
        return count
    }
}
let counter = makeCounter()
print(counter())  // 1
print(counter())  // 2

    26. 尾随闭包

    🌟 知识点讲解

    • 尾随闭包语法：当闭包是函数最后一个参数时，可将闭包写在函数括号后面
    • 省略括号：若函数只有闭包一个参数且使用尾随闭包，可省略函数调用的括号
    • 多个闭包：Swift 5.3+支持多个尾随闭包语法，使代码更清晰
    • 逃逸闭包：使用@escaping标注的闭包在函数返回后才执行，如异步回调
    swift
    // 尾随闭包
func performOperation(_ a: Int, _ b: Int, operation: (Int, Int) -> Int) -> Int {
    return operation(a, b)
}

// 普通写法
let result1 = performOperation(3, 4, operation: { $0 + $1 })

// 尾随闭包写法
let result2 = performOperation(3, 4) { $0 + $1 }
print(result2)  // 7

// 省略括号
let digits = [1, 2, 3, 4, 5]
let transformed = digits.map { $0 * 2 }
print(transformed)  // [2, 4, 6, 8, 10]

func loadData(success: () -> Void, failure: () -> Void) {
    success()
}
loadData {
    print("加载成功")
} failure: {
    print("加载失败")
}

var completionHandlers: [() -> Void] = []
func withEscaping(completion: @escaping () -> Void) {
    completionHandlers.append(completion)
}

    非尾参数使用尾随闭包：闭包不是最后一个参数时不能使用尾随闭包语法

    32. 协议protocol

    🌟 知识点讲解

    • 协议定义：使用protocol关键字定义蓝图，规定方法、属性和下标的要求

    34. 错误处理

    🌟 知识点讲解

    • 错误协议：自定义错误类型遵循Error协议，通常用枚举表示不同错误情况
    • 抛出错误：使用throw关键字抛出错误，函数需标记throws
    • 处理方式：do-catch捕获、try?转为可选值、try!强制执行（可能崩溃）
    • defer语句：无论是否抛出错误都会执行的清理代码，类似finally
    swift
    // 定义错误类型
enum VendingError: Error {
    case outOfStock
    case insufficientFunds(coinsNeeded: Int)
}

// 抛出错误的函数
struct Item {
    var name: String
    var price: Int
    var count: Int
}

class VendingMachine {
    var inventory = ["Candy": Item(name: "Candy", price: 5, count: 3)]
    var coinsDeposited = 0
    
    func vend(itemNamed name: String) throws -> Item {
        guard let item = inventory[name] else {
            throw VendingError.outOfStock
        }
        guard item.count > 0 else {
            throw VendingError.outOfStock
        }
        guard coinsDeposited >= item.price else {
            throw VendingError.insufficientFunds(coinsNeeded: item.price - coinsDeposited)
        }
        coinsDeposited -= item.price
        return item
    }
}

// do-catch处理
let machine = VendingMachine()
machine.coinsDeposited = 3
do {
    let item = try machine.vend(itemNamed: "Candy")
    print("购买成功: \(item.name)")
} catch VendingError.insufficientFunds(let coinsNeeded) {
    print("余额不足，还需\(coinsNeeded)枚硬币")
} catch {
    print("其他错误: \(error)")
}

// try? 转为可选值
if let item = try? machine.vend(itemNamed: "Candy") {
    print(item.name)
} else {
    print("购买失败")
}

// defer语句
func processFile() throws {
    print("打开文件")
    defer { print("关闭文件") }
    print("处理文件")
}
try processFile()
// 输出: 打开文件 → 处理文件 → 关闭文件

    调用throws函数不用try：try是必须的，否则编译错误

extension Gradable {
    func average() throws -> Double {
        guard !scores.isEmpty else { return 0 }
        let total = scores.values.reduce(0, +)
        let avg = total / Double(scores.count)
        guard avg >= 0 && avg <= 100 else {
            throw GradeError.invalidScore(avg)
        }
        return avg
    }
    
    func grade() throws -> String {
        let avg = try average()
        switch avg {
        case 90...100: return "A"
        case 80..<90:  return "B"
        case 70..<80:  return "C"
        case 60..<70:  return "D"
        default:       return "F"
        }
    }
}

    }
    
    mutating func addScore(_ score: Double, for subject: String) throws {
        guard score >= 0 && score <= 100 else {
            throw GradeError.invalidScore(score)
        }
        scores[subject] = score
    }
}

// 班级管理器
class ClassManager {
    private var students: [String: Student] = [:]
    
    func enroll(_ student: Student) {
        students[student.id] = student
    }
    
    func student(for id: String) -> Student? {
        return students[id]
    }
    
    func updateScore(studentId: String, score: Double, subject: String) throws {
        guard students[studentId] != nil else {
            throw GradeError.studentNotFound(studentId)
        }
        try students[studentId]?.addScore(score, for: subject)
    }
    
    func ranking() -> [(Student, Double)] {
        return students.values.compactMap { student in
            guard let avg = try? student.average() else { return nil }
            return (student, avg)
        }.sorted { $0.1 > $1.1 }
    }
}

    }
    
    print("\n排名:")

        print("\(index + 1). \(item.0.name): \(String(format: "%.1f", item.1))")
