    1. 入门介绍与环境搭建

    Go是Google开发的开源编程语言，于2009年正式发布。它结合了C语言的高性能和Python的开发效率，特别适合构建高并发的网络服务和分布式系统。

    🌟 Go语言的特点

    • 简洁高效：语法简洁，编译速度快，执行效率高
    • 并发支持：原生支持并发编程，goroutine轻量级线程
    • 内存管理：自动垃圾回收，无需手动管理内存
    • 跨平台：支持Windows、Mac、Linux等多种操作系统
    • 静态类型：编译时类型检查，保证代码安全
    • 丰富标准库：强大的标准库，开箱即用

    📦 安装Go

    从 golang.org/dl 下载并安装Go。安装完成后，可以通过以下命令验证安装：

    bash 
    go version
# 输出示例: go version go1.21.0 darwin/amd64

    🚀 第一个Go程序

    创建一个名为 hello.go 的文件：

    go 
    package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

    运行程序：

    bash 
    go run hello.go
# 输出: Hello, World!

    📁 Go工作空间

    Go使用GOPATH作为工作空间，包含三个目录：

    • src/ - 源代码文件
    • pkg/ - 编译后的包文件
    • bin/ - 编译后的可执行文件

    ❌ 常见错误

    缺少main包：package main 是可执行程序必须的包声明。

    缺少main函数：每个可执行程序必须有一个 func main() 函数作为入口。

    未使用的导入：Go编译器不允许导入未使用的包。

    2. 基本语法

    📦 变量声明

    Go提供多种变量声明方式：

    go 
    package main

import "fmt"

func main() {
    // 方式1: 使用var关键字
    var name string = "Go"
    var age int = 13
    
    // 方式2: 类型推断
    var message = "Hello"
    
    // 方式3: 短变量声明（只能在函数内部使用）
    count := 100
    
    // 声明多个变量
    var a, b, c int
    var x, y = 10, "test"
    
    fmt.Println(name, age, message, count, a, b, c, x, y)
}

    📊 数据类型

    Go的基本数据类型：

    • 数值类型：int, int8, int16, int32, int64, uint, uint8, uint16, uint32, uint64, float32, float64
    • 布尔类型：bool (true/false)
    • 字符串：string
    • 复数类型：complex64, complex128
    go 
    package main

import "fmt"

func main() {
    var i int = 42
    var f float64 = 3.14
    var b bool = true
    var s string = "hello"
    
    fmt.Printf("int: %d\n", i)
    fmt.Printf("float: %f\n", f)
    fmt.Printf("bool: %t\n", b)
    fmt.Printf("string: %s\n", s)
}

    🔢 常量

    使用 const 声明常量：

    ❌ 常见错误

    类型不匹配：Go是强类型语言，不能直接将int赋值给float。

    未使用的变量：Go编译器不允许声明但未使用的变量。

    3. 流程控制

    🔀 if-else语句

    Go的if语句不需要括号，但大括号必须有：

    go 
    package main

import "fmt"

func main() {
    age := 18
    
    if age >= 18 {
        fmt.Println("成年人")
    } else if age >= 12 {
        fmt.Println("青少年")
    } else {
        fmt.Println("儿童")
    }
    
    // if可以包含初始化语句
    if score := 85; score >= 90 {
        fmt.Println("优秀")
    } else {
        fmt.Println("继续努力")
    }
}

    🔄 for循环

    Go只有for循环，没有while和do-while：

    go 
    package main

import "fmt"

func main() {
    // 基本for循环
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
    
    // 类似while循环
    count := 0
    for count < 5 {
        fmt.Println("count:", count)
        count++
    }
    
    // 无限循环
    // for {
    //     fmt.Println("无限循环")
    // }
    
    // range遍历
    fruits := []string{"apple", "banana", "orange"}
    for index, value := range fruits {
        fmt.Printf("索引: %d, 值: %s\n", index, value)
    }
}

    🔀 switch语句

    Go的switch非常灵活，可以比较任意类型：

    go 
    package main

import "fmt"

func main() {
    day := 3
    
    switch day {
    case 1:
        fmt.Println("星期一")
    case 2:
        fmt.Println("星期二")
    case 3, 4, 5:
        fmt.Println("工作日")
    default:
        fmt.Println("周末")
    }
    
    // switch可以不带表达式
    score := 85
    switch {
    case score >= 90:
        fmt.Println("优秀")
    case score >= 80:
        fmt.Println("良好")
    default:
        fmt.Println("加油")
    }
}

    ⏭️ break和continue

    go 
    package main

import "fmt"

func main() {
    // break: 跳出循环
    for i := 0; i < 10; i++ {
        if i == 5 {
            break
        }
        fmt.Println(i)
    }
    
    // continue: 跳过当前迭代
    for i := 0; i < 10; i++ {
        if i%2 == 0 {
            continue
        }
        fmt.Println("奇数:", i)
    }
}

    ❌ 常见错误

    if条件加括号：if (x > 0) 虽然不会报错，但Go风格不推荐。

    for循环缺少大括号：for i := 0; i < 10; i++ fmt.Println(i) 会报错。

    switch的case不需要break：Go的switch在每个case末尾自动break，需要穿透时使用 fallthrough。

    4. 数组与切片

    📦 数组

    数组是固定长度的相同类型元素序列：

    go 
    package main

import "fmt"

func main() {
    // 声明数组
    var arr [5]int
    arr[0] = 1
    arr[1] = 2
    
    // 初始化数组
    numbers := [5]int{1, 2, 3, 4, 5}
    
    // 省略长度，由初始化值决定
    primes := [...]int{2, 3, 5, 7, 11}
    
    // 二维数组
    matrix := [2][3]int{
        {1, 2, 3},
        {4, 5, 6},
    }
    
    fmt.Println(arr)
    fmt.Println(numbers)
    fmt.Println(primes)
    fmt.Println(matrix)
}

    🔪 切片

    切片是数组的引用，长度可变：

    go 
    package main

import "fmt"

func main() {
    // 从数组创建切片
    arr := [5]int{1, 2, 3, 4, 5}
    slice := arr[1:4] // [2, 3, 4]
    
    // 直接创建切片
    s := []int{1, 2, 3}
    
    // 使用make创建切片
    s1 := make([]int, 5)      // 长度5，容量5
    s2 := make([]int, 5, 10)  // 长度5，容量10
    
    fmt.Println(slice)
    fmt.Println(s)
    fmt.Println(s1, len(s1), cap(s1))
    fmt.Println(s2, len(s2), cap(s2))
}

    ⚡ 切片操作

    go 
    package main

import "fmt"

func main() {
    s := []int{1, 2, 3, 4, 5}
    
    // 添加元素
    s = append(s, 6, 7, 8)
    fmt.Println("append:", s)
    
    // 切片
    fmt.Println("slice[1:3]:", s[1:3])
    fmt.Println("slice[:3]:", s[:3])
    fmt.Println("slice[2:]:", s[2:])
    
    // 复制
    s2 := make([]int, len(s))
    copy(s2, s)
    fmt.Println("copy:", s2)
    
    // 删除元素（删除索引为2的元素）
    s = append(s[:2], s[3:]...)
    fmt.Println("delete:", s)
}
    💡 提示：切片是引用类型，多个切片可能指向同一个底层数组。修改切片会影响共享同一底层数组的其他切片。

    ❌ 常见错误

    数组越界访问：arr[10] 当数组长度小于11时会报错。

    切片与数组混淆：数组传递是值拷贝，切片传递是引用传递。

    append后未赋值：append(s, 1) 不会修改原切片，必须赋值：s = append(s, 1)。

    📝 练习题

    1. 切片和数组的主要区别是什么？
    数组长度固定，切片长度可变；数组是值类型，切片是引用类型
    2. 写代码创建一个切片，添加元素1,2,3，然后删除中间的元素2
    package main import "fmt" func main() { s := []int{1, 2, 3} s = append(s[:1], s[2:]...) fmt.Println(s) // [1 3] }

    5. 函数

    🔧 函数定义

    Go函数的定义格式：

    go 
    package main

import "fmt"

// 无参数无返回值
func greet() {
    fmt.Println("Hello!")
}

// 有参数有返回值
func add(a int, b int) int {
    return a + b
}

// 多返回值
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, fmt.Errorf("cannot divide by zero")
    }
    return a / b, nil
}

// 命名返回值
func calculate(a, b int) (sum, diff int) {
    sum = a + b
    diff = a - b
    return // 直接返回命名变量
}

func main() {
    greet()
    fmt.Println(add(3, 5))
    
    result, err := divide(10, 2)
    if err != nil {
        fmt.Println("Error:", err)
    } else {
        fmt.Println(result)
    }
    
    s, d := calculate(10, 3)
    fmt.Println("sum:", s, "diff:", d)
}

    📦 可变参数

    go 
    package main

import "fmt"

func sum(nums ...int) int {
    total := 0
    for _, num := range nums {
        total += num
    }
    return total
}

func main() {
    fmt.Println(sum(1, 2, 3))
    fmt.Println(sum(10, 20, 30, 40))
    
    // 传递切片
    arr := []int{1, 2, 3, 4, 5}
    fmt.Println(sum(arr...))
}

    🔗 函数作为参数

    go 
    package main

import "fmt"

func apply(nums []int, f func(int) int) []int {
    result := make([]int, len(nums))
    for i, v := range nums {
        result[i] = f(v)
    }
    return result
}

func main() {
    nums := []int{1, 2, 3, 4}
    
    // 使用匿名函数
    doubled := apply(nums, func(x int) int {
        return x * 2
    })
    fmt.Println(doubled)
    
    squared := apply(nums, func(x int) int {
        return x * x
    })
    fmt.Println(squared)
}

    🔄 递归函数

    go 
    package main

import "fmt"

func factorial(n int) int {
    if n == 0 {
        return 1
    }
    return n * factorial(n-1)
}

func fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    return fibonacci(n-1) + fibonacci(n-2)
}

func main() {
    fmt.Println("5! =", factorial(5))
    fmt.Println("fib(10) =", fibonacci(10))
}

    ❌ 常见错误

    缺少返回值：函数声明有返回值但没有return语句会报错。

    参数类型不匹配：传递的参数类型必须与函数声明一致。

    未使用的参数：Go编译器不允许函数中有未使用的参数（但可以用下划线忽略）。

    6. 结构体与方法

    📦 结构体定义

    结构体是自定义的复合类型：

    go 
    package main

import "fmt"

// 定义结构体
type Person struct {
    Name string
    Age  int
    City string
}

func main() {
    // 创建结构体实例
    p1 := Person{Name: "Alice", Age: 25, City: "Beijing"}
    p2 := Person{"Bob", 30, "Shanghai"} // 按顺序赋值
    
    // 访问字段
    fmt.Println(p1.Name, p1.Age)
    
    // 修改字段
    p2.Age = 31
    fmt.Println(p2)
}

    🔧 结构体方法

    方法是与特定类型关联的函数：

    go 
    package main

import "fmt"

type Person struct {
    Name string
    Age  int
}

// 值接收者方法
func (p Person) Greet() string {
    return fmt.Sprintf("Hello, my name is %s, I'm %d years old", p.Name, p.Age)
}

// 指针接收者方法（可以修改结构体）
func (p *Person) Birthday() {
    p.Age++
}

func main() {
    p := Person{Name: "Alice", Age: 25}
    fmt.Println(p.Greet())
    
    p.Birthday()
    fmt.Println("After birthday:", p.Age)
}

    🔗 结构体嵌套

    go 
    package main

import "fmt"

type Address struct {
    Street string
    City   string
    Zip    string
}

type Person struct {
    Name    string
    Age     int
    Address Address // 嵌套结构体
}

func main() {
    p := Person{
        Name: "Alice",
        Age:  25,
        Address: Address{
            Street: "123 Main St",
            City:   "Beijing",
            Zip:    "100000",
        },
    }
    
    // 访问嵌套字段
    fmt.Println(p.Address.City)
    
    // 结构体可以直接比较（所有字段可比较）
    p2 := Person{Name: "Alice", Age: 25, Address: p.Address}
    fmt.Println(p == p2) // true
}

    🔄 构造函数

    go 
    package main

import "fmt"

type Rectangle struct {
    Width  float64
    Height float64
}

// 构造函数
func NewRectangle(width, height float64) *Rectangle {
    return &Rectangle{Width: width, Height: height}
}

func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}

func (r Rectangle) Perimeter() float64 {
    return 2 * (r.Width + r.Height)
}

func main() {
    rect := NewRectangle(10, 5)
    fmt.Println("面积:", rect.Area())
    fmt.Println("周长:", rect.Perimeter())
}

    ❌ 常见错误

    使用值接收者修改结构体：值接收者方法中修改结构体不会影响原实例，需要使用指针接收者。

    字段名大小写错误：大写字母开头的字段是导出的，可以在包外访问；小写字母开头的字段是私有的。

    结构体比较：只有所有字段都可比较的结构体才能比较。

    📝 练习题

    1. 值接收者和指针接收者的区别是什么？
    值接收者复制结构体，不会修改原实例；指针接收者引用结构体，可以修改原实例

    7. 接口

    📦 接口定义

    接口是方法签名的集合，定义了对象的行为：

    go 
    package main

import "fmt"

// 定义接口
type Shape interface {
    Area() float64
    Perimeter() float64
}

// Circle实现Shape接口
type Circle struct {
    Radius float64
}

func (c Circle) Area() float64 {
    return 3.14159 * c.Radius * c.Radius
}

func (c Circle) Perimeter() float64 {
    return 2 * 3.14159 * c.Radius
}

// Rectangle实现Shape接口
type Rectangle struct {
    Width  float64
    Height float64
}

func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}

func (r Rectangle) Perimeter() float64 {
    return 2 * (r.Width + r.Height)
}

func main() {
    var s Shape
    
    s = Circle{Radius: 5}
    fmt.Println("Circle:", s.Area(), s.Perimeter())
    
    s = Rectangle{Width: 10, Height: 5}
    fmt.Println("Rectangle:", s.Area(), s.Perimeter())
}

    🔀 类型断言

    使用类型断言判断接口值的具体类型：

    go 
    package main

import "fmt"

type Shape interface {
    Area() float64
}

type Circle struct {
    Radius float64
}

func (c Circle) Area() float64 {
    return 3.14159 * c.Radius * c.Radius
}

func describe(s Shape) {
    fmt.Printf("Type: %T, Value: %v\n", s, s)
    
    // 类型断言
    if c, ok := s.(Circle); ok {
        fmt.Printf("Circle Radius: %.2f\n", c.Radius)
    }
    
    // switch类型断言
    switch v := s.(type) {
    case Circle:
        fmt.Printf("It's a circle with radius %.2f\n", v.Radius)
    default:
        fmt.Println("Unknown type")
    }
}

func main() {
    c := Circle{Radius: 5}
    describe(c)
}

    📦 空接口

    空接口可以存储任何类型的值：

    go 
    package main

import "fmt"

func printValue(i interface{}) {
    fmt.Printf("Type: %T, Value: %v\n", i, i)
}

func main() {
    printValue(42)
    printValue("hello")
    printValue(3.14)
    printValue(true)
    
    // 存储到切片中
    var values []interface{}
    values = append(values, 10, "test", 3.14, false)
    
    for _, v := range values {
        switch v.(type) {
        case int:
            fmt.Println("Integer:", v)
        case string:
            fmt.Println("String:", v)
        case float64:
            fmt.Println("Float:", v)
        default:
            fmt.Println("Other:", v)
        }
    }
}
    💡 提示：Go的接口是隐式实现的，不需要显式声明。只要类型实现了接口的所有方法，就自动实现了该接口。

    ❌ 常见错误

    接口方法未完全实现：类型必须实现接口的所有方法，否则不能赋值给接口变量。

    空接口的误用：过度使用空接口会丧失类型安全性，应谨慎使用。

    类型断言失败：未检查ok标志就进行类型断言会导致panic。

    📝 练习题

    1. Go的接口是显式实现还是隐式实现？
    隐式实现，不需要显式声明
    2. 定义一个Animal接口，包含Speak方法，然后定义Dog和Cat结构体实现该接口
    package main import "fmt" type Animal interface { Speak() string } type Dog struct{} func (d Dog) Speak() string { return "Woof!" } type Cat struct{} func (c Cat) Speak() string { return "Meow!" } func main() { var a Animal a = Dog{} fmt.Println(a.Speak()) a = Cat{} fmt.Println(a.Speak()) }

    8. 并发编程(goroutine)

    🚀 Goroutine基础

    Goroutine是Go语言的轻量级线程，由Go运行时管理：

    go 
    package main

import (
    "fmt"
    "time"
)

func sayHello(name string) {
    for i := 0; i < 5; i++ {
        fmt.Printf("Hello, %s! %d\n", name, i)
        time.Sleep(100 * time.Millisecond)
    }
}

func main() {
    // 启动goroutine
    go sayHello("Alice")
    go sayHello("Bob")
    
    // 等待goroutine完成
    time.Sleep(1 * time.Second)
    fmt.Println("Done!")
}

    🔗 Channel通道

    Channel用于goroutine之间的通信：

    go 
    package main

import "fmt"

func producer(ch chan<- int) {
    for i := 0; i < 5; i++ {
        ch <- i // 发送数据
        fmt.Println("Produced:", i)
    }
    close(ch) // 关闭通道
}

func consumer(ch <-chan int) {
    for num := range ch { // 接收数据
        fmt.Println("Consumed:", num)
    }
}

func main() {
    // 创建通道
    ch := make(chan int)
    
    go producer(ch)
    go consumer(ch)
    
    // 等待完成
    var input string
    fmt.Scanln(&input)
}

    📦 Sync包

    使用sync.WaitGroup等待多个goroutine完成：

    go 
    package main

import (
    "fmt"
    "sync"
    "time"
)

func worker(id int, wg *sync.WaitGroup) {
    defer wg.Done() // 通知完成
    
    fmt.Printf("Worker %d starting\n", id)
    time.Sleep(time.Second)
    fmt.Printf("Worker %d done\n", id)
}

func main() {
    var wg sync.WaitGroup
    
    for i := 1; i <= 3; i++ {
        wg.Add(1) // 添加任务
        go worker(i, &wg)
    }
    
    wg.Wait() // 等待所有任务完成
    fmt.Println("All workers done")
}

    ❌ 常见错误

    主线程提前退出：goroutine还没执行完，main函数就退出了，导致goroutine被终止。

    死锁：两个goroutine互相等待对方的数据，导致死锁。

    关闭已关闭的通道：重复关闭通道会导致panic。

    向已关闭的通道发送数据：会导致panic。

        return
    }

}

    10. 综合实战

    🎯 项目介绍

    在这个综合实战中，我们将创建一个简单的HTTP服务器，实现用户管理功能，包括：用户列表、添加用户、查询用户等。

    go 
    package main

import (
    "encoding/json"
    "fmt"
    "log"
    "net/http"
    "strconv"
    "sync"
)

// User结构体
type User struct {
    ID   int    `json:"id"`
    Name string `json:"name"`
    Age  int    `json:"age"`
    City string `json:"city"`
}

// 用户存储
var users = []User{
    {ID: 1, Name: "Alice", Age: 25, City: "Beijing"},
    {ID: 2, Name: "Bob", Age: 30, City: "Shanghai"},
}
var nextID = 3
var mu sync.Mutex

// 获取所有用户
func getUsers(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(users)
}

// 获取单个用户
func getUser(w http.ResponseWriter, r *http.Request) {
    idStr := r.URL.Path[len("/users/"):]
    id, err := strconv.Atoi(idStr)
    if err != nil {
        http.Error(w, "Invalid user ID", http.StatusBadRequest)
        return
    }
    
    for _, user := range users {
        if user.ID == id {
            w.Header().Set("Content-Type", "application/json")
            json.NewEncoder(w).Encode(user)
            return
        }
    }
    
    http.Error(w, "User not found", http.StatusNotFound)
}

// 创建用户
func createUser(w http.ResponseWriter, r *http.Request) {
    var user User
    if err := json.NewDecoder(r.Body).Decode(&user); err != nil {
        http.Error(w, "Invalid request body", http.StatusBadRequest)
        return
    }
    
    mu.Lock()
    user.ID = nextID
    nextID++
    users = append(users, user)
    mu.Unlock()
    
    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(http.StatusCreated)
    json.NewEncoder(w).Encode(user)
}

// 更新用户
func updateUser(w http.ResponseWriter, r *http.Request) {
    idStr := r.URL.Path[len("/users/"):]
    id, err := strconv.Atoi(idStr)
    if err != nil {
        http.Error(w, "Invalid user ID", http.StatusBadRequest)
        return
    }
    
    var updatedUser User
    if err := json.NewDecoder(r.Body).Decode(&updatedUser); err != nil {
        http.Error(w, "Invalid request body", http.StatusBadRequest)
        return
    }
    
    mu.Lock()
    for i, user := range users {
        if user.ID == id {
            users[i].Name = updatedUser.Name
            users[i].Age = updatedUser.Age
            users[i].City = updatedUser.City
            w.Header().Set("Content-Type", "application/json")
            json.NewEncoder(w).Encode(users[i])
            mu.Unlock()
            return
        }
    }
    mu.Unlock()
    
    http.Error(w, "User not found", http.StatusNotFound)
}

// 删除用户
func deleteUser(w http.ResponseWriter, r *http.Request) {
    idStr := r.URL.Path[len("/users/"):]
    id, err := strconv.Atoi(idStr)
    if err != nil {
        http.Error(w, "Invalid user ID", http.StatusBadRequest)
        return
    }
    
    mu.Lock()
    for i, user := range users {
        if user.ID == id {
            users = append(users[:i], users[i+1:]...)
            w.WriteHeader(http.StatusNoContent)
            mu.Unlock()
            return
        }
    }
    mu.Unlock()
    
    http.Error(w, "User not found", http.StatusNotFound)
}

func main() {
    http.HandleFunc("/users", func(w http.ResponseWriter, r *http.Request) {
        switch r.Method {
        case "GET":
            getUsers(w, r)
        case "POST":
            createUser(w, r)
        default:
            http.Error(w, "Method not allowed", http.StatusMethodNotAllowed)
        }
    })
    
    http.HandleFunc("/users/", func(w http.ResponseWriter, r *http.Request) {
        switch r.Method {
        case "GET":
            getUser(w, r)
        case "PUT":
            updateUser(w, r)
        case "DELETE":
            deleteUser(w, r)
        default:
            http.Error(w, "Method not allowed", http.StatusMethodNotAllowed)
        }
    })
    
    fmt.Println("Server starting on :8080...")
    log.Fatal(http.ListenAndServe(":8080", nil))
}
