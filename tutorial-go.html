    1. Go语言简介

    🌟 知识点讲解

    • Go的诞生：由Google的Robert Griesemer、Rob Pike和Ken Thompson于2007年设计，2009年开源，2012年发布1.0
    • 设计目标：解决大型软件工程的开发效率，兼顾编译型性能和解释型开发速度
    • 核心特性：静态类型、编译型、原生并发、垃圾回收、简洁语法、快速编译
    • 应用领域：云计算(Docker/K8s)、微服务、网络编程、DevOps、区块链
    • 语言哲学：少即是多——减少特性降低复杂度，没有继承、没有异常
    go
    package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
    fmt.Println("欢迎学习Go语言教程！")
    version := "1.22"
    fmt.Printf("当前Go版本: %s\n", version)
}

    ❌ 常见错误

    混淆Go与C++：Go是全新设计的语言，没有类和继承

    误以为Go是脚本语言：Go是编译型语言，但go run可简化过程

    📝 练习题

    Go语言的三位创始人分别是谁？
    Robert Griesemer、Rob Pike、Ken Thompson

    2. 开发环境配置

    🌟 知识点讲解

    • 安装Go：从golang.org/dl下载安装包，推荐最新稳定版
    • 环境变量：GOPATH工作空间、GOROOT安装路径、PATH添加bin目录
    • Go模块：1.11引入模块系统，1.16后默认，使用go.mod管理依赖
    • IDE推荐：VS Code+Go扩展、GoLand、Vim+vim-go
    go
    # 检查Go版本
go version

# 初始化模块
mkdir myproject && cd myproject
go mod init github.com/yourname/myproject

# 添加依赖
go get github.com/gin-gonic/gin

# 运行和编译
go run main.go
go build -o myapp main.go

# 格式化和检查
go fmt ./...
vet ./...

    ❌ 常见错误

    忘记go mod init：没有模块无法管理依赖

    未设置GOPATH：模块模式下可任意目录开发

    3. 第一个Go程序

    🌟 知识点讲解

    • package声明：每个Go文件必须以package开头，可执行程序入口包必须是main
    • import导入：导入标准库或第三方包，未使用的导入会编译错误
    • main函数：程序入口，无参数无返回值，必须在main包中
    • 语句规则：不需要分号，左大括号不能单独一行
    go
    package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
    name := "Go语言"
    fmt.Printf("欢迎学习%s！\n", name)
    a, b := 10, 20
    fmt.Printf("%d + %d = %d\n", a, b, a+b)
}

    ❌ 常见错误

    左大括号换行：func main()\n{是错误的

    未使用的导入：编译失败

    📝 练习题

    编写程序输出你的名字和年龄
    package main import "fmt" func main() { name := "张三" age := 25 fmt.Printf("我叫%s，今年%d岁\n", name, age) }

    4. 包与导入

    🌟 知识点讲解

    • 包的概念：Go通过package组织代码，同目录文件必须同包
    • 可见性规则：大写字母开头=导出(public)，小写=未导出(private)
    • 导入方式：单行、多行、别名、点导入
    • 常用标准库：fmt、os、strings、net/http
    go

    "fmt"
    "math"
    "strings"
    "time"
)

    fmt.Println("Hello")
    fmt.Println("圆周率:", math.Pi)
    fmt.Println(strings.ToUpper("hello go"))

}

    ❌ 常见错误

    循环导入：Go不允许循环依赖

    导入未使用：编译错误，可用_空白标识符

    5. 变量声明

    🌟 知识点讲解

    6. 常量与iota

    🌟 知识点讲解

    • const声明：常量在编译时确定，不能修改
    • iota枚举器：const块中每新增一行iota加1，从0开始
    • iota技巧：跳过值_ = iota、位运算1 << iota
    • 无类型常量：常量可不指定类型，需要时自动转换
    go
    package main

import "fmt"

const Pi = 3.14159

const (
    Sunday    = iota // 0
    Monday           // 1
    Tuesday          // 2
    Wednesday        // 3
    Thursday         // 4
    Friday           // 5
    Saturday         // 6
)

const (
    ReadPermission   = 1 << iota // 1
    WritePermission              // 2
    ExecutePermission            // 4
)

func main() {
    fmt.Println("Pi =", Pi)
    fmt.Println("Monday =", Monday)
    fmt.Println("All =", ReadPermission|WritePermission|ExecutePermission)
}

    ❌ 常见错误

    const块外使用iota：编译错误

    常量用函数赋值：const t = time.Now()错误

    📝 练习题

    使用iota定义文件权限常量：读(1)、写(2)、执行(4)
    const ( Read = 1 << iota // 1 Write // 2 Execute // 4 )

    7. 基本数据类型

    🌟 知识点讲解

    • 整数类型：int8~int64、uint8~uint64，int/uint与平台相关

    8. 类型转换

    🌟 知识点讲解

    • 显式转换：Go没有隐式转换，必须T(v)
    • 数值间转换：int(float64)，注意精度丢失
    • 字符串转换：string([]byte)、strconv.Itoa等
    • strconv包：Atoi、Itoa、FormatFloat等
    go
    package main

import (
    "fmt"
    "strconv"
)

func main() {
    var i int = 42
    var f float64 = float64(i)
    var u uint = uint(f)
    fmt.Println(i, f, u)
    s := strconv.Itoa(123)
    n, _ := strconv.Atoi("456")
    fmt.Println(s, n)
    data := []byte("Hello")
    fmt.Println(string(data))
}

    ❌ 常见错误

    隐式转换：var i int = 3.14不允许

    精度丢失：大数转换需注意

    10. 比较与逻辑运算符

    🌟 知识点讲解

    • 比较运算符：==、!=、<、>、<=、>=
    • 逻辑运算符：&&与、||或、!非
    • 短路求值：&&左false不执行右，||左true不执行右
    • 可比较类型：相同类型才能比较

    } else {
        fmt.Println("短路保护")
    }
}

    11. if语句

    🌟 知识点讲解

    • 基本if：条件不需要括号，大括号必须有
    • if-else：else必须与if的}在同一行

    12. switch语句

    🌟 知识点讲解

    • 自动break：默认每个case自动终止
    • fallthrough：继续执行下一个case
    • 多值匹配：case 1, 2, 3:
    • 无条件switch：相当于if-else链
    go
    package main

import "fmt"

func main() {
    day := "Wednesday"
    switch day {
    case "Monday":
        fmt.Println("周一")
    case "Wednesday", "Thursday":
        fmt.Println("周中")
    default:
        fmt.Println("其他")
    }
    score := 85
    switch {
    case score >= 90:
        fmt.Println("优秀")
    case score >= 80:
        fmt.Println("良好")
    }
}

    ❌ 常见错误

    📝 练习题

    13. for循环

    🌟 知识点讲解

    • Go唯一循环：只有for，没有while
    • 经典for：for init; cond; post {}
    • 条件for：for cond {}相当于while
    • range迭代：for i, v := range col {}
    go
    package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Print(i, " ")

    for n < 100 { n *= 2 }
    fmt.Println("n =", n)
    nums := []int{10, 20, 30}
    for i, v := range nums {
        fmt.Printf("%d: %d\n", i, v)
    }
}

    14. break与continue

    🌟 知识点讲解

    • break：终止当前循环
    • continue：跳过当前迭代
    • 标签break：break Label跳出外层
    • 标签continue：continue Label继续外层
    go
    package main

import "fmt"

func main() {
    for i := 0; i < 10; i++ {
        if i == 5 { break }
        fmt.Print(i, " ")
    }
    fmt.Println()
    for i := 0; i < 10; i++ {
        if i%2 == 0 { continue }
        fmt.Print(i, " ")
    }
    fmt.Println()
outer:
    for i := 0; i < 3; i++ {
        for j := 0; j < 3; j++ {
            if i == 1 && j == 1 { break outer }
            fmt.Printf("(%d,%d) ", i, j)
        }
    }
}

    ❌ 常见错误

    标签未定义：编译错误

    误用goto：不推荐

    📝 练习题

    找出第一个能被7整除且大于100的数

    17. 切片高级操作

    🌟 知识点讲解

    18. map映射

    🌟 知识点讲解

    • 创建：make(map[K]V)、map[K]V{}
    • 操作：存取、delete、检查val, ok := m[key]

    • nil map：写入会panic，需make初始化
    go
    package main

func main() {
    scores := make(map[string]int)

    for name, score := range scores {
        fmt.Printf("%s: %d\n", name, score)
    }
    fmt.Println(colors)
}

    ❌ 常见错误

    35. 综合实战

    🌟 知识点讲解

    • 项目设计：综合运用struct、interface、goroutine等
    • 并发模式：worker pool、pipeline
    • HTTP服务：net/http或gin框架
    • 项目结构：cmd/、pkg/、internal/标准布局
    go
    package main

import (
    "fmt"
    "sync"
)

type Task interface {
    Execute() error
    Name() string
}

type PrintTask struct{ Message string }
func (t *PrintTask) Execute() error {
    fmt.Println("执行:", t.Message)
    return nil
}
func (t *PrintTask) Name() string { return t.Message }

func workerPool(tasks []Task, workerCount int) {
    taskCh := make(chan Task, len(tasks))
    var wg sync.WaitGroup
    for i := 0; i < workerCount; i++ {
        wg.Add(1)
        go func(id int) {
            defer wg.Done()
            for task := range taskCh {
                fmt.Printf("Worker %d 处理 %s\n", id, task.Name())
                task.Execute()
            }
        }(i)
    }
    for _, t := range tasks { taskCh <- t }
    close(taskCh)
    wg.Wait()
}

func main() {
    tasks := []Task{
        &PrintTask{"任务A"},
        &PrintTask{"任务B"},
        &PrintTask{"任务C"},
        &PrintTask{"任务D"},
        &PrintTask{"任务E"},
    }
    workerPool(tasks, 3)
    fmt.Println("所有任务完成！")
}
