    1. Go语言简介

    🌟 知识点讲解

    • Go的诞生：由Google的Robert Griesemer、Rob Pike和Ken Thompson于2007年设计，2009年开源，2012年发布1.0版本
    • 设计目标：解决大型软件工程的开发效率问题，兼顾编译型语言的性能和解释型语言的开发速度
    • 核心特性：静态类型、编译型语言、原生并发支持、垃圾回收、简洁语法、快速编译
    • 应用领域：云计算(Docker/Kubernetes)、微服务架构、网络编程、DevOps工具、区块链
    • 语言哲学：少即是多——通过减少语言特性来降低复杂度，没有继承、没有异常处理机制
    go
    package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
    fmt.Println("欢迎学习Go语言教程！")
    version := "1.22"
    fmt.Printf("当前Go版本: %s\n", version)
}

    ❌ 常见错误

    混淆Go与C++：Go是全新设计的语言，没有类和继承概念

    误以为Go是脚本语言：Go是编译型语言，但go run命令可简化编译运行过程

    📝 练习题

    Go语言的三位创始人分别是谁？
    Robert Griesemer、Rob Pike、Ken Thompson

    2. 开发环境配置

    🌟 知识点讲解

    • 安装Go：从golang.org/dl下载安装包，推荐安装最新稳定版
    • 环境变量：GOPATH工作空间、GOROOT安装路径、PATH添加bin目录
    • Go模块：1.11版本引入模块系统，1.16版本后默认启用，使用go.mod管理依赖
    • IDE推荐：VS Code+Go扩展、GoLand、Vim+vim-go插件
    bash
    # 检查Go版本
go version

# 初始化模块
mkdir myproject && cd myproject
go mod init github.com/yourname/myproject

# 添加依赖
go get github.com/gin-gonic/gin

# 运行和编译
go run main.go
go build -o myapp main.go

# 格式化和检查
go fmt ./...
go vet ./...

    ❌ 常见错误

    忘记执行go mod init：没有模块无法管理依赖

    未设置GOPATH：在模块模式下可以在任意目录开发，不需要GOPATH

    📝 练习题

    3. 第一个Go程序

    🌟 知识点讲解

    • package声明：每个Go文件必须以package开头，可执行程序的入口包必须是main
    • import导入：导入标准库或第三方包，未使用的导入会导致编译错误
    • main函数：程序入口函数，无参数无返回值，必须定义在main包中
    • 语句规则：不需要分号结尾，左大括号不能单独放在一行
    go
    package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
    name := "Go语言"
    fmt.Printf("欢迎学习%s！\n", name)
    a, b := 10, 20
    fmt.Printf("%d + %d = %d\n", a, b, a+b)
}

    ❌ 常见错误

    左大括号换行：func main()\n{是错误的写法

    导入的包未使用：会导致编译失败

    📝 练习题

    编写程序输出你的名字和年龄
    package main import "fmt" func main() { name := "张三" age := 25 fmt.Printf("我叫%s，今年%d岁\n", name, age) }

    4. 包与导入

    🌟 知识点讲解

    • 包的概念：Go通过package组织代码，同一目录下的文件必须属于同一个包
    • 可见性规则：大写字母开头的标识符是导出(public)的，小写字母开头是未导出(private)的
    • 导入方式：单行导入、多行导入、别名导入、点导入
    • 常用标准库：fmt、os、strings、net/http等
    go
    package main

import (
    "fmt"
    "math"
    "strings"
    "time"
)

func main() {
    fmt.Println("Hello")
    fmt.Println("圆周率:", math.Pi)
    fmt.Println(strings.ToUpper("hello go"))
    fmt.Println(time.Now().Format("2006-01-02 15:04:05"))
}

    ❌ 常见错误

    循环导入：Go不允许包之间存在循环依赖

    导入了未使用的包：编译错误，可用_空白标识符忽略

    📝 练习题

    Go中如何区分导出和未导出的标识符？
    大写字母开头为导出(public)，小写字母开头为未导出(private)

    5. 变量声明

    🌟 知识点讲解

    • var声明：var name type = value，类型可以省略，由编译器推断
    • 短变量声明：name := value，只能在函数内部使用
    • 零值机制：数值类型为0、字符串为""、布尔为false、引用类型为nil
    • 多重赋值：a, b := 1, 2，支持交换操作a, b = b, a
    go
    package main

import "fmt"

func main() {
    var name string = "Go"
    var age int = 14
    var city = "Beijing"
    var count int      // 零值: 0
    var message string // 零值: ""
    var flag bool      // 零值: false
    lang := "Go语言"
    version := 1.22
    a, b := 10, 20
    a, b = b, a
    fmt.Println(name, age, city, count, flag, lang, version, a, b)
}

    ❌ 常见错误

    在函数外部使用:=：必须使用var声明

    声明变量后未使用：编译错误

    📝 练习题

    6. 常量与iota

    🌟 知识点讲解

    • const声明：常量在编译时确定，程序运行时不能修改
    • iota枚举器：在const块中每新增一行iota值加1，从0开始
    • iota技巧：跳过值使用_ = iota、位运算1 << iota
    • 无类型常量：常量可以不指定类型，需要时自动转换
    go
    package main

import "fmt"

const Pi = 3.14159

const (
    Sunday    = iota // 0
    Monday           // 1
    Tuesday          // 2
    Wednesday        // 3
    Thursday         // 4
    Friday           // 5
    Saturday         // 6
)

const (
    ReadPermission   = 1 << iota // 1
    WritePermission              // 2
    ExecutePermission            // 4
)

func main() {
    fmt.Println("Pi =", Pi)
    fmt.Println("Monday =", Monday)
    fmt.Println("All =", ReadPermission|WritePermission|ExecutePermission)
}

    ❌ 常见错误

    在const块外使用iota：编译错误

    常量用函数赋值：const t = time.Now()是错误的，因为time.Now()在运行时执行

    📝 练习题

    使用iota定义文件权限常量：读(1)、写(2)、执行(4)
    const ( Read = 1 << iota // 1 Write // 2 Execute // 4 )

    7. 基本数据类型

    🌟 知识点讲解

    • 整数类型：int8~int64、uint8~uint64，int/uint与平台相关
    • 浮点类型：float32(约6位精度)、float64(约15位精度)
    • 布尔类型：bool，只有true和false两个值
    • 字符串类型：string，UTF-8编码，不可变
    • byte与rune：byte是uint8的别名，rune是int32的别名
    go
    package main

import "fmt"

func main() {
    var i int = 42
    var f64 float64 = 3.14159265358979
    var b bool = true
    s := "Hello, 世界"
    ch := '中'
    fmt.Printf("字符: %c, Unicode: %U\n", ch, ch)
    fmt.Println(i, f64, b, s, len(s))
}

    ❌ 常见错误

    整数溢出：int8最大127，超出范围会编译错误或运行时panic

    字符串长度：len("中文")返回6(字节数)而不是2(字符数)

    8. 类型转换

    🌟 知识点讲解

    • 显式转换：Go没有隐式类型转换，必须使用T(v)语法
    • 数值间转换：int(float64)，注意精度丢失问题
    • 字符串转换：string([]byte)、strconv.Itoa等
    • strconv包：提供Atoi、Itoa、FormatFloat等转换函数
    go
    package main

import (
    "fmt"
    "strconv"
)

func main() {
    var i int = 42
    var f float64 = float64(i)
    var u uint = uint(f)
    fmt.Println(i, f, u)
    s := strconv.Itoa(123)
    n, _ := strconv.Atoi("456")
    fmt.Println(s, n)
    data := []byte("Hello")
    fmt.Println(string(data))
}

    ❌ 常见错误

    隐式转换：var i int = 3.14不允许

    精度丢失：大数值转换时需要注意

    📝 练习题

    将字符串"123"转换为整数并加1输出
    n, _ := strconv.Atoi("123") fmt.Println(n + 1) // 124

    9. 算术运算符

    🌟 知识点讲解

    • 基本运算：+、-、*、/、%
    • 整数除法：整数相除向下取整，5/2=2
    • 自增自减：只有i++和i--，是语句不是表达式
    • 位运算：&与、|或、^异或、<<左移、>>右移

}

    10. 比较与逻辑运算符

    🌟 知识点讲解

    • 比较运算符：==、!=、<、>、<=、>=
    • 逻辑运算符：&&与、||或、!非
    • 短路求值：&&左边为false不执行右边，||左边为true不执行右边
    • 可比较类型：只有相同类型的值才能比较
    go
    package main

import "fmt"

func main() {
    a, b := 10, 20
    fmt.Println("a < b:", a < b)
    x, y := true, false
    fmt.Println("x && y:", x && y)
    fmt.Println("x || y:", x || y)
    val := 0
    if val != 0 && 100/val > 10 {
        fmt.Println("成立")
    } else {
        fmt.Println("短路保护")
    }
}

    ❌ 常见错误

    比较不同类型的值：编译错误

    slice和map不能用==比较

    📝 练习题

    Go中逻辑运算符短路求值的含义？
    &&左边为false则右边不执行；||左边为true则右边不执行

    11. if语句

    🌟 知识点讲解

    • 基本if：条件不需要括号，大括号必须存在
    • if-else：else必须与if的}在同一行
    • if-else if-else：多条件链式判断
    • 初始化语句：if stmt; condition {}，变量作用域仅限于if块内
    go
    package main

import "fmt"

func main() {
    score := 85
    if score >= 90 {
        fmt.Println("优秀")
    } else if score >= 80 {
        fmt.Println("良好")
    } else {
        fmt.Println("及格")
    }
    if num := 42; num%2 == 0 {
        fmt.Printf("%d是偶数\n", num)
    }
}

    ❌ 常见错误

    else换行：必须与}在同一行

    条件加括号：虽然语法正确但不推荐

    📝 练习题

    使用if初始化语句判断偶数

    12. switch语句

    🌟 知识点讲解

    • 自动break：默认每个case执行完自动终止
    • fallthrough：继续执行下一个case
    • 多值匹配：case 1, 2, 3:
    • 无条件switch：相当于if-else链
    go
    package main

import "fmt"

func main() {
    day := "Wednesday"
    switch day {
    case "Monday":
        fmt.Println("周一")
    case "Wednesday", "Thursday":
        fmt.Println("周中")
    default:
        fmt.Println("其他")
    }
    score := 85
    switch {
    case score >= 90:
        fmt.Println("优秀")
    case score >= 80:
        fmt.Println("良好")
    }
}

    ❌ 常见错误

    忘记fallthrough：默认不会穿透到下一个case

    case类型不匹配：编译错误

    📝 练习题

    用switch判断月份所属季节
    switch month { case 3, 4, 5: fmt.Println("春季") case 6, 7, 8: fmt.Println("夏季") case 9, 10, 11: fmt.Println("秋季") case 12, 1, 2: fmt.Println("冬季") }

    13. for循环

    🌟 知识点讲解

    • Go唯一循环：只有for循环，没有while和do-while
    • 经典for：for init; cond; post {}
    • 条件for：for cond {}相当于while循环
    • range迭代：for i, v := range col {}
    go
    package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Print(i, " ")
    }
    fmt.Println()
    n := 1
    for n < 100 { n *= 2 }
    fmt.Println("n =", n)
    nums := []int{10, 20, 30}
    for i, v := range nums {
        fmt.Printf("%d: %d\n", i, v)
    }
}

    ❌ 常见错误

    range返回值是副本：修改v不影响原切片

    循环变量捕获问题（Go 1.22之前版本）

    14. break与continue

    🌟 知识点讲解

    • break：终止当前循环
    • continue：跳过当前迭代
    • 标签break：break Label跳出外层循环
    • 标签continue：continue Label继续外层循环
    go
    package main

import "fmt"

func main() {
    for i := 0; i < 10; i++ {
        if i == 5 { break }
        fmt.Print(i, " ")
    }
    fmt.Println()
    for i := 0; i < 10; i++ {
        if i%2 == 0 { continue }
        fmt.Print(i, " ")
    }
    fmt.Println()
outer:
    for i := 0; i < 3; i++ {
        for j := 0; j < 3; j++ {
            if i == 1 && j == 1 { break outer }
            fmt.Printf("(%d,%d) ", i, j)
        }
    }
}

    ❌ 常见错误

    使用未定义的标签：编译错误

    误用goto：不推荐使用

    📝 练习题

    找出第一个能被7整除且大于100的数
    for i := 101; ; i++ { if i%7 == 0 { fmt.Println(i) // 105 break } }

    15. 数组

    🌟 知识点讲解

    • 声明：var arr [5]int，长度是类型的一部分
    • 初始化：[5]int{1,2,3}、[...]int{1,2,3}
    • 值类型：赋值和传参时复制整个数组

}

    16. 切片基础

    🌟 知识点讲解

    • 切片vs数组：切片是动态长度的引用类型，底层指向数组
    • 创建：make([]T, len, cap)、[]T{}、从数组切片

    17. 切片高级操作

    🌟 知识点讲解

    • 切片表达式：s[low:high:max]控制容量
    • copy函数：copy(dst, src)
    • 删除元素：append(s[:i], s[i+1:]...)
    • 二维切片：[][]int需要逐行初始化
    go
    package main

import "fmt"

func main() {
    s := []int{1, 2, 3, 4, 5}
    s = append(s[:2], s[3:]...)
    fmt.Println(s)
    src := []int{1, 2, 3}
    dst := make([]int, len(src))
    copy(dst, src)
    fmt.Println(dst)
    matrix := make([][]int, 3)
    for i := range matrix {
        matrix[i] = make([]int, 4)
    }
    fmt.Println(matrix)
}

    ❌ 常见错误

    append可能触发扩容导致引用失效

    18. map映射

    🌟 知识点讲解

    • 创建：make(map[K]V)、map[K]V{}
    • 操作：存取、delete、检查val, ok := m[key]
    • 遍历：使用range，遍历顺序不确定
    • nil map：写入会panic，需要先make初始化
    go
    package main

import "fmt"

func main() {
    scores := make(map[string]int)
    scores["张三"] = 95
    scores["李四"] = 88
    colors := map[string]string{"red": "红色", "green": "绿色"}
    val, ok := scores["赵六"]
    fmt.Println(val, ok) // 0 false
    delete(scores, "李四")
    for name, score := range scores {
        fmt.Printf("%s: %d\n", name, score)
    }
    fmt.Println(colors)
}

    ❌ 常见错误

    向nil map写入会panic

    map非线程安全，并发需要sync.Mutex或sync.Map

    📝 练习题

    统计字符串中每个字符出现的次数
    s := "hello" count := make(map[rune]int) for _, ch := range s { count[ch]++ } fmt.Println(count)

    22. panic与recover

    🌟 知识点讲解

    • panic：运行时异常，终止程序执行
    • recover：捕获panic，恢复程序运行
    • 使用场景：严重错误处理、测试框架
    • 最佳实践：不用panic处理预期错误
    go
    package main

import "fmt"

func safeDivide(a, b int) (result int, err error) {
    defer func() {
        if r := recover(); r != nil {
            err = fmt.Errorf("panic: %v", r)
        }
    }()
    if b == 0 {
        panic("division by zero")
    }
    return a / b, nil
}

func main() {
    res, err := safeDivide(10, 0)
    if err != nil {
        fmt.Println("Error:", err)
    } else {
        fmt.Println("Result:", res)
    }
}

    ❌ 常见错误

    recover必须在defer中调用

    不要滥用panic处理业务错误

    📝 练习题

    编写安全函数捕获数组越界panic

            fmt.Println(msg1)
        case msg2 := <-ch2:
            fmt.Println(msg2)
        case <-time.After(3 * time.Second):
            fmt.Println("Timeout!")

    }

    • RLock/RUnlock：读锁可重入
    go
    package main

    "fmt"
    "sync"
)

    mu.Unlock()

func main() {

    wg.Wait()

    go
    // 实战项目：简单的待办事项API
package main

import (
    "encoding/json"
    "net/http"
    "sync"
)

type Todo struct {
    ID    int    `json:"id"`
    Title string `json:"title"`
    Done  bool   `json:"done"`
}

var todos []Todo
var mu sync.Mutex
var nextID = 1

func getTodos(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(todos)
}

func addTodo(w http.ResponseWriter, r *http.Request) {
    var todo Todo
    json.NewDecoder(r.Body).Decode(&todo)
    mu.Lock()
    todo.ID = nextID
    nextID++
    todos = append(todos, todo)
    mu.Unlock()
    w.WriteHeader(http.StatusCreated)
    json.NewEncoder(w).Encode(todo)
}

func main() {
    http.HandleFunc("/todos", getTodos)
    http.HandleFunc("/todos/add", addTodo)
    http.ListenAndServe(":8080", nil)
}
