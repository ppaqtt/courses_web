JavaScript 完整教程

从零开始学习 JavaScript，掌握现代 Web 开发核心技术

01

JavaScript 简介

▼

知识点

  • JavaScript 是一种脚本语言，用于为网页添加交互性
  • 1995年由 Brendan Eich 在 Netscape 公司创建
  • JavaScript 可以在浏览器和服务器端（Node.js）运行
  • 是 Web 开发三大核心技术之一（HTML、CSS、JavaScript）
  • 支持面向对象、函数式和事件驱动编程

代码示例

hello-world.js 
// 第一个 JavaScript 程序
console.log('Hello, JavaScript!');

// 在网页中显示消息
alert('欢迎学习 JavaScript！');

// 向 HTML 页面写入内容
document.write('<h1>Hello, World!</h1>');

常见错误

❌ 常见错误：拼写错误

将 console.log 拼写成 console.Log 或 Console.log，JavaScript 是大小写敏感的语言。

练习题

  1. 使用 console.log() 输出你的名字
  2. 使用 alert() 显示一条欢迎消息
  3. 使用 document.write() 在页面上显示当前日期

02

变量与数据类型

▼

知识点

  • 变量用于存储数据值，使用 var、let、const 声明
  • let 和 const 是 ES6 引入的块级作用域变量
  • const 声明的变量不能重新赋值
  • 基本数据类型：string、number、boolean、null、undefined、symbol
  • typeof 运算符用于检测数据类型

代码示例

variables.js 
// 使用 var 声明变量（函数作用域）
var name = '张三';
var age = 25;

// 使用 let 声明变量（块级作用域）
let isStudent = true;
let score = 95.5;

// 使用 const 声明常量
const PI = 3.14159;
const COLOR = '#667eea';

// 检测数据类型
console.log(typeof name);    // string
console.log(typeof age);     // number
console.log(typeof isStudent); // boolean

常见错误

❌ 常见错误：重复声明变量

使用 let 或 const 重复声明同一变量会抛出错误。应该避免在同一作用域内重复声明变量。

练习题

  1. 声明一个变量存储你的年龄，并输出其类型
  2. 声明一个常量存储圆周率
  3. 声明一个变量存储你的邮箱，然后修改它的值

03

运算符

▼

知识点

  • 算术运算符：+、-、*、/、%、++、--
  • 赋值运算符：=、+=、-=、*=、/=
  • 比较运算符：==、===、!=、!==、>、<、>=、<=
  • 逻辑运算符：&&、||、!
  • 三元运算符：条件 ? 表达式1 : 表达式2

代码示例

operators.js 
// 算术运算符
let a = 10;
let b = 3;
console.log(a + b);   // 13
console.log(a % b);   // 1
console.log(a++);     // 10（后自增）
console.log(++a);     // 12（前自增）

// 比较运算符
console.log(5 == '5');    // true（宽松相等）
console.log(5 === '5');   // false（严格相等）

// 逻辑运算符
let x = true;
let y = false;
console.log(x && y);  // false
console.log(x || y);  // true
console.log(!x);      // false

// 三元运算符
let score = 85;
let result = score >= 60 ? '及格' : '不及格';
console.log(result);  // 及格

常见错误

❌ 常见错误：混淆 == 和 ===

使用 == 会进行类型转换，可能导致意外结果。推荐使用 === 进行严格相等比较。

练习题

  1. 计算两个数的和、差、积、商
  2. 使用三元运算符判断一个数是正数还是负数
  3. 使用逻辑运算符判断年份是否为闰年

流程控制 - if-else

▼

知识点

  • if 语句用于条件判断
  • else 语句处理条件不满足的情况
  • else if 用于多个条件判断
  • 条件表达式返回布尔值（true/false）
  • 代码块使用花括号 {} 包裹

代码示例

if-else.js 
// 简单 if 语句
let age = 18;
if (age >= 18) {
    console.log('成年人');
}

// if-else 语句
let score = 75;
if (score >= 60) {
    console.log('及格');
} else {
    console.log('不及格');
}

// else if 多条件判断
let grade = 'B';
if (grade === 'A') {
    console.log('优秀');
} else if (grade === 'B') {
    console.log('良好');
} else if (grade === 'C') {
    console.log('中等');
} else {
    console.log('需努力');
}

常见错误

❌ 常见错误：使用 = 代替 ==

在条件判断中使用赋值运算符 = 而不是比较运算符 ==，这会导致条件永远为真（除非赋值为 falsy 值）。

练习题

  1. 判断一个数是否为偶数
  2. 根据分数输出对应的等级（90以上A，80-89B，70-79C，60-69D，60以下F）
  3. 判断年份是否为闰年

流程控制 - switch

▼

知识点

  • switch 语句用于多条件分支判断
  • 每个 case 对应一个分支
  • break 语句用于跳出 switch
  • default 处理未匹配的情况
  • switch 使用严格相等（===）比较

代码示例

switch.js 
// switch 基本用法
let day = 3;
switch (day) {
    case 1:
        console.log('星期一');
        break;
    case 2:
        console.log('星期二');
        break;
    case 3:
        console.log('星期三');
        break;
    default:
        console.log('其他日期');
}

// 多个 case 共享代码块
let month = 2;
switch (month) {
    case 1:
    case 3:
    case 5:
    case 7:
    case 8:
    case 10:
    case 12:
        console.log('31天');
        break;
    case 4:
    case 6:
    case 9:
    case 11:
        console.log('30天');
        break;
    case 2:
        console.log('28或29天');
        break;
}

常见错误

❌ 常见错误：忘记 break 语句

如果忘记在 case 末尾添加 break，程序会继续执行下一个 case，这称为"贯穿"现象。

练习题

  1. 使用 switch 根据月份输出季节
  2. 使用 switch 根据成绩等级输出评语
  3. 使用 switch 判断星期几并输出对应的中文名称

06

循环 - for

▼

知识点

  • for 循环由初始化、条件、更新三部分组成
  • 语法：for(初始化; 条件; 更新) { 代码块 }
  • break 跳出循环
  • continue 跳过当前迭代
  • 循环变量通常用 i、j、k 命名

代码示例

for-loop.js 
// 基本 for 循环
for (let i = 0; i < 5; i++) {
    console.log('计数: ' + i);
}

// 遍历数组
let fruits = ['苹果', '香蕉', '橙子'];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

// 使用 break 跳出循环
for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break;  // 当 i 等于 5 时跳出
    }
    console.log(i);
}

// 使用 continue 跳过迭代
for (let i = 0; i < 10; i++) {
    if (i % 2 === 0) {
        continue;  // 跳过偶数
    }
    console.log(i);
}

常见错误

❌ 常见错误：无限循环

忘记更新循环变量或条件永远为真会导致无限循环，使浏览器卡死。

练习题

  1. 使用 for 循环输出 1 到 100 的所有数
  2. 计算 1 到 100 的和
  3. 输出 100 以内的所有质数

07

循环 - while 与 do-while

▼

知识点

  • while 循环：先判断条件，再执行代码
  • do-while 循环：先执行代码，再判断条件
  • do-while 至少执行一次
  • while 可能一次都不执行
  • 同样支持 break 和 continue

代码示例

while-loop.js 
// while 循环
let count = 0;
while (count < 5) {
    console.log('计数: ' + count);
    count++;
}

// do-while 循环
let num = 0;
do {
    console.log('数字: ' + num);
    num++;
} while (num < 5);

// 求阶乘
let n = 5;
let factorial = 1;
while (n > 0) {
    factorial *= n;
    n--;
}
console.log('5的阶乘: ' + factorial);

// 猜数字游戏
let target = 7;
let guess;
do {
    guess = parseInt(prompt('猜一个1-10的数字'));
} while (guess !== target);
console.log('猜对了！');

常见错误

❌ 常见错误：忘记更新循环变量

在 while 或 do-while 循环中忘记更新控制变量会导致无限循环。

练习题

  1. 使用 while 循环计算 1 到 50 的和
  2. 使用 do-while 循环输出 20 到 1 的倒序
  3. 使用 while 循环实现斐波那契数列前20项

函数基础

▼

知识点

  • 函数是可重复使用的代码块
  • 使用 function 关键字定义函数
  • 函数可以有参数和返回值
  • 函数调用时传递实参
  • 函数可以没有参数和返回值

代码示例

functions.js 
// 无参数无返回值函数
function greet() {
    console.log('Hello!');
}
greet();

// 有参数函数
function sayHello(name) {
    console.log('Hello, ' + name + '!');
}
sayHello('张三');

// 有返回值函数
function add(a, b) {
    return a + b;
}
let result = add(3, 5);
console.log(result);  // 8

// 多参数函数
function calculate(a, b, operator) {
    switch(operator) {
        case '+': return a + b;
        case '-': return a - b;
        case '*': return a * b;
        case '/': return a / b;
        default: return '无效运算符';
    }
}
console.log(calculate(10, 2, '*'));  // 20

常见错误

❌ 常见错误：函数名拼写错误

调用函数时拼写错误会导致"is not defined"错误。确保函数名和调用时一致。

练习题

  1. 定义一个函数计算圆的面积
  2. 定义一个函数判断一个数是否为质数
  3. 定义一个函数将华氏温度转换为摄氏温度

09

函数参数与返回值

▼

知识点

  • 参数分为形参和实参
  • ES6 支持默认参数值
  • 使用 arguments 对象访问所有参数
  • return 语句可以返回任意类型的值
  • 没有 return 的函数返回 undefined

代码示例

parameters.js 
// 默认参数
function greet(name = 'Guest') {
    console.log('Hello, ' + name);
}
greet();          // Hello, Guest
greet('李四');    // Hello, 李四

// 使用 arguments 对象
function sum() {
    let total = 0;
    for (let i = 0; i < arguments.length; i++) {
        total += arguments[i];
    }
    return total;
}
console.log(sum(1, 2, 3, 4));  // 10

// 返回对象
function createUser(name, age) {
    return {
        name: name,
        age: age,
        createdAt: new Date()
    };
}
let user = createUser('王五', 30);
console.log(user);

// 返回函数
function createMultiplier(n) {
    return function(x) {
        return x * n;
    };
}
let double = createMultiplier(2);
console.log(double(5));  // 10

常见错误

❌ 常见错误：参数数量不匹配

传递的参数少于函数定义的参数时，多余的参数为 undefined。传递过多参数则会被忽略。

练习题

  1. 定义一个函数，计算任意数量数字的平均值
  2. 定义一个函数，返回数组中的最大值和最小值
  3. 定义一个函数，接收姓名和年龄，返回问候语

作用域

▼

知识点

  • 全局作用域：在函数外部声明的变量
  • 局部作用域：在函数内部声明的变量
  • 块级作用域：ES6 的 let/const 声明
  • 变量提升：var 声明的变量会提升到作用域顶部
  • 作用域链：内部函数可以访问外部函数的变量

代码示例

scope.js 
// 全局变量
let globalVar = '全局';

function outer() {
    // 函数作用域变量
    let outerVar = '外部';
    
    function inner() {
        // 内部函数可以访问外部变量
        let innerVar = '内部';
        console.log(globalVar);   // 全局
        console.log(outerVar);    // 外部
        console.log(innerVar);    // 内部
    }
    
    inner();
    // console.log(innerVar); // 错误：无法访问内部变量
}

outer();

// 块级作用域
if (true) {
    let blockVar = '块级';
    console.log(blockVar);  // 块级
}
// console.log(blockVar); // 错误：let 声明的变量只在块内有效

// 变量提升
console.log(hoistedVar);  // undefined（变量被提升，但值未初始化）
var hoistedVar = '提升';

常见错误

❌ 常见错误：误用全局变量

在函数内部未使用 var/let/const 声明的变量会成为全局变量，这可能导致意外的副作用。

练习题

  1. 解释全局作用域和局部作用域的区别
  2. 为什么使用 let 比 var 更好？
  3. 编写代码演示变量提升现象

数组

▼

知识点

  • 数组是有序的数据集合
  • 使用 [] 创建数组
  • 通过索引访问数组元素（从0开始）
  • length 属性获取数组长度
  • 常用方法：push、pop、shift、unshift、slice、splice

代码示例

arrays.js 
// 创建数组
let fruits = ['苹果', '香蕉', '橙子'];
let numbers = [1, 2, 3, 4, 5];
let mixed = [1, 'hello', true, null];

// 访问数组元素
console.log(fruits[0]);  // 苹果
console.log(fruits[2]);  // 橙子

// 修改数组元素
fruits[1] = '葡萄';
console.log(fruits);  // ['苹果', '葡萄', '橙子']

// 数组长度
console.log(fruits.length);  // 3

// 添加元素
fruits.push('西瓜');      // 末尾添加
fruits.unshift('草莓');   // 开头添加
console.log(fruits);

// 删除元素
fruits.pop();       // 删除末尾
fruits.shift();     // 删除开头
console.log(fruits);

// 切片
let sliced = fruits.slice(1, 3);
console.log(sliced);  // ['葡萄', '橙子']

// 拼接
fruits.splice(1, 0, '芒果', '菠萝');
console.log(fruits);

❌ 常见错误：数组索引越界

访问不存在的数组索引会返回 undefined，而不是报错。注意数组索引从0开始。

练习题

  3. 找出数组中的最大值

12

数组方法

▼

知识点

  • forEach：遍历数组
  • map：转换数组元素
  • filter：过滤数组元素
  • reduce：累加数组元素
  • find/findIndex：查找元素
  • sort：排序数组

代码示例

array-methods.js 
let numbers = [1, 2, 3, 4, 5];

// forEach 遍历
numbers.forEach(function(num, index) {
    console.log(index + ': ' + num);
});

// map 转换
let doubled = numbers.map(function(num) {
    return num * 2;
});
console.log(doubled);  // [2, 4, 6, 8, 10]

// filter 过滤
let evens = numbers.filter(function(num) {
    return num % 2 === 0;
});
console.log(evens);  // [2, 4]

// reduce 累加
let sum = numbers.reduce(function(acc, num) {
    return acc + num;
}, 0);
console.log(sum);  // 15

// find 查找
let found = numbers.find(function(num) {
    return num > 3;
});
console.log(found);  // 4

// sort 排序
let words = ['banana', 'apple', 'cherry'];
words.sort();
console.log(words);  // ['apple', 'banana', 'cherry']

常见错误

❌ 常见错误：sort 方法默认按字符串排序

数组的 sort() 方法默认将元素转换为字符串进行比较，数字排序需要提供比较函数。

练习题

  1. 使用 map 将数组中的每个元素平方
  2. 使用 filter 过滤出大于 10 的数字
  3. 使用 reduce 计算数组中所有元素的乘积

对象

▼

知识点

  • 对象是键值对的集合
  • 使用 {} 创建对象
  • 通过点号或方括号访问属性
  • 可以添加、修改、删除属性
  • for...in 遍历对象属性

代码示例

objects.js 
// 创建对象
let person = {
    name: '张三',
    age: 25,
    isStudent: true,
    hobbies: ['阅读', '编程']
};

// 访问属性
console.log(person.name);           // 张三
console.log(person['age']);         // 25
console.log(person.hobbies[0]);     // 阅读

// 修改属性
person.age = 26;
person['isStudent'] = false;

// 添加属性
person.email = 'zhangsan@example.com';
person.address = {
    city: '北京',
    street: '朝阳区'
};

// 删除属性
delete person.isStudent;

// 遍历对象
for (let key in person) {
    console.log(key + ': ' + person[key]);
}

// 检查属性是否存在
console.log('name' in person);      // true
console.log('phone' in person);     // false

常见错误

❌ 常见错误：访问未定义的属性

访问对象不存在的属性会返回 undefined，而不是报错。使用 in 运算符检查属性是否存在。

练习题

字符串方法

▼

知识点

  • length：获取字符串长度
  • charAt/charCodeAt：获取字符
  • indexOf/lastIndexOf：查找字符位置
  • slice/substring：截取字符串
  • toUpperCase/toLowerCase：转换大小写
  • trim：去除首尾空格
  • split：分割字符串
  • replace：替换字符串

代码示例

string-methods.js 
let str = '  Hello, JavaScript!  ';

// 字符串长度
console.log(str.length);  // 24

// 获取字符
console.log(str.charAt(2));    // H
console.log(str.charCodeAt(2)); // 72

// 查找位置
console.log(str.indexOf('o'));    // 4
console.log(str.lastIndexOf('a')); // 14

// 截取字符串
console.log(str.slice(2, 7));     // Hello
console.log(str.substring(2, 7)); // Hello

// 转换大小写
console.log(str.toUpperCase());   // HELLO, JAVASCRIPT!
console.log(str.toLowerCase());   // hello, javascript!

// 去除空格
console.log(str.trim());          // Hello, JavaScript!

// 分割字符串
let words = str.trim().split(' ');
console.log(words);  // ['Hello,', 'JavaScript!']

// 替换
console.log(str.replace('JavaScript', 'World'));

// 模板字符串
let name = '张三';
let greeting = `你好，${name}！今天是${new Date().getFullYear()}年`;
console.log(greeting);

常见错误

❌ 常见错误：字符串是不可变的

JavaScript 中的字符串是不可变的，所有字符串方法都返回新字符串，不会修改原字符串。

练习题

  1. 统计字符串中某个字符出现的次数
  2. 反转一个字符串
  3. 检查字符串是否为回文

日期对象

知识点

  • getFullYear：获取年份

  • getDate：获取日期（1-31）

date.js 
// 创建日期对象

    let timeout;

            setTimeout(() => inThrottle = false, limit);
        }
    };
}
