    1. C++简介与历史

    C++是一种通用的、静态类型的、编译式编程语言，由Bjarne Stroustrup于1979年在贝尔实验室开始设计，作为C语言的扩展（最初被称为"C with Classes"）。

    🌟 核心知识点

    • C++的诞生：1979年Bjarne Stroustrup在C语言基础上增加类、虚函数等面向对象特性，1983年正式命名为C++
    • 标准演进：C++98（首个标准）→ C++03 → C++11（重大革新）→ C++14 → C++17 → C++20 → C++23
    • 多范式支持：C++同时支持过程式编程、面向对象编程、泛型编程和函数式编程
    • 应用领域：系统软件、游戏开发、嵌入式系统、高性能计算、金融建模、编译器开发
    • C与C++关系：C++是C的超集（几乎所有合法C代码也是合法C++代码），但C++增加了类型检查、类、模板等
    cpp
    #include <iostream>
using namespace std;

int main() {
    cout << "Hello, C++!" << endl;
    cout << "C++ is a powerful language!" << endl;
    return 0;
}

    ❌ 常见错误

    忘记 #include <iostream> 会导致编译错误，因为 cout 和 endl 定义在该头文件中。

    修正：始终确保包含所需的头文件。

    4. 注释与命名空间

    注释用于解释代码，命名空间用于组织代码和避免命名冲突。

    🌟 核心知识点

    • 单行注释：以 // 开头，到行尾结束
    • 多行注释：以 /* 开头，*/ 结束，可跨越多行
    • 命名空间定义：namespace MyNS { ... } 封装相关代码
    • using声明：using namespace std; 或 using std::cout;
    cpp
    #include <iostream>

namespace Math {
    double PI = 3.14159;
    int square(int x) { return x * x; }
}

int main() {
    // 单行注释示例
    cout << "PI = " << Math::PI << endl;
    /* 多行注释示例

    cout << "5^2 = " << Math::square(5) << endl;
    return 0;
}

    ❌ 常见错误

    多行注释不能嵌套：/* 外层 /* 内层 */ 继续外层 */ 会导致编译错误，因为第一个 */ 就结束了注释。

    5. 基本数据类型

    C++提供了丰富的内置数据类型来存储不同种类的数据。

    🌟 核心知识点

    • 整数类型：short(2字节)、int(4字节)、long(4/8字节)、long long(8字节)，各有signed/unsigned版本
    • 浮点类型：float(4字节，约7位精度)、double(8字节，约15位精度)、long double
    • 字符类型：char(1字节)、wchar_t、char16_t、char32_t
    • 布尔类型：bool，值为true或false
    • sizeof运算符：获取类型或变量占用的字节数
    cpp
    #include <iostream>
using namespace std;

int main() {
    int a = 42;
    double pi = 3.14159;
    char ch = 'A';
    bool flag = true;

    ❌ 常见错误

    6. 变量与常量

    变量是存储数据的命名内存空间，常量是不可修改的固定值。

    🌟 核心知识点

    • 变量声明与初始化：int x = 10;（拷贝初始化）、int x{10};（列表初始化，C++11）
    • const常量：const int MAX = 100; 必须在声明时初始化，之后不能修改
    • constexpr（C++11）：编译期常量，constexpr int SIZE = 10;
    • 宏定义常量：#define PI 3.14（不推荐，优先使用const）
    cpp
    #include <iostream>
using namespace std;

int main() {
    int age = 25;           // 变量
    const double PI = 3.14159;  // const常量
    constexpr int SIZE = 10;    // constexpr常量

    // age = 30;  // OK: 变量可以修改
    // PI = 3.0;  // 错误: const不能修改

    cout << "age = " << age << endl;
    cout << "PI = " << PI << endl;
    cout << "SIZE = " << SIZE << endl;
    return 0;
}

    ❌ 常见错误

    未初始化的局部变量使用未定义行为：int x; cout << x; 输出不确定的值。应始终初始化变量。

    7. 类型转换

    C++支持多种类型转换方式，包括隐式转换和显式转换。

    🌟 核心知识点

    • 隐式转换：小类型→大类型自动转换，如int→double
    • C风格转换：(double)x，功能强大但不够安全
    • static_cast：编译期转换，static_cast<double>(x)，最常用
    • dynamic_cast：运行时多态类型转换，用于继承体系中的向下转型
    • const_cast/reinterpret_cast：去除const限定/重新解释位模式
    cpp
    #include <iostream>
using namespace std;

int main() {
    int a = 7, b = 2;

    // 隐式转换: 先转double再除
    double r1 = a / b;          // 3 (整数除法)
    double r2 = (double)a / b;  // 3.5 (C风格)
    double r3 = static_cast<double>(a) / b; // 3.5 (C++风格)

    cout << "r1 = " << r1 << endl;
    cout << "r2 = " << r2 << endl;
    cout << "r3 = " << r3 << endl;
    return 0;
}

    ❌ 常见错误

    整数除法丢失精度：double r = 7 / 2; 结果为3.0而非3.5，因为7/2先做整数除法得3，再隐式转为3.0。修正：double r = 7.0 / 2;

    8. 算术运算符

    算术运算符用于执行基本的数学运算。

    🌟 核心知识点

    • 基本运算：+（加）、-（减）、*（乘）、/（除）、%（取模）
    • 自增自减：++（自增）、--（自减），分前缀和后缀形式
    • 复合赋值：+=、-=、*=、/=、%=
    • 运算符优先级：* / % 高于 + -，可用括号改变
    cpp
    #include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;
    cout << "a + b = " << a + b << endl;  // 13
    cout << "a - b = " << a - b << endl;  // 7
    cout << "a * b = " << a * b << endl;  // 30
    cout << "a / b = " << a / b << endl;  // 3
    cout << "a % b = " << a % b << endl;  // 1

    int c = 5;
    cout << "c++ = " << c++ << endl;  // 5 (后缀: 先用后加)
    cout << "++c = " << ++c << endl;  // 7 (前缀: 先加后用)
    return 0;
}

    ❌ 常见错误

    对浮点数使用取模运算符：double x = 5.5 % 2; 编译错误，%只能用于整数操作数。

    9. 关系与逻辑运算符

    关系运算符比较值的大小，逻辑运算符组合布尔表达式。

    🌟 核心知识点

    • 关系运算符：==、!=、<、>、<=、>=，返回bool值
    • 逻辑运算符：&&（与）、||（或）、!（非）
    • 短路求值：&&左边为false时不再计算右边；||左边为true时不再计算右边
    • C++20宇宙飞船运算符：<=> 三路比较运算符
    cpp
    #include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;
    cout << boolalpha;  // 输出true/false而非1/0
    cout << "a > b: " << (a > b) << endl;
    cout << "a == b: " << (a == b) << endl;
    cout << "(a < b) && (b > 0): " << ((a < b) && (b > 0)) << endl;
    cout << "(a > b) || (b > 0): " << ((a > b) || (b > 0)) << endl;
    return 0;
}

    ❌ 常见错误

    将 == 写成 =：if (x = 5) 是赋值而非比较，结果始终为true（非零值）。应写 if (x == 5)。

    10. 位运算符

    位运算符直接操作二进制位，常用于底层编程和性能优化。

    🌟 核心知识点

    • 位运算符：&（按位与）、|（按位或）、^（按位异或）、~（按位取反）
    • 移位运算符：<<（左移，相当于×2）、>>（右移，相当于÷2）
    • 常用技巧：判断奇偶 n & 1、交换变量 a ^= b; b ^= a; a ^= b;
    • 位掩码：设置/清除/翻转特定位，广泛用于标志位操作
    cpp
    #include <iostream>
using namespace std;

int main() {
    unsigned int a = 0b1100;  // 12
    unsigned int b = 0b1010;  // 10

    cout << "a & b = " << (a & b) << endl;   // 8  (1000)
    cout << "a | b = " << (a | b) << endl;   // 14 (1110)
    cout << "a ^ b = " << (a ^ b) << endl;   // 6  (0110)
    cout << "a << 1 = " << (a << 1) << endl; // 24 (左移1位)
    return 0;
}

    ❌ 常见错误

    对有符号数使用右移：负数右移行为由实现定义，可能进行算术右移（补1）或逻辑右移（补0）。建议对位运算使用无符号类型。

    11. if-else语句

    if-else是最基本的条件判断语句，根据条件执行不同的代码分支。

    🌟 核心知识点

    • 基本形式：if (条件) { ... } else { ... }
    • 多重分支：if-else if-else链，从上到下依次判断
    • 嵌套if：if内部可以嵌套另一个if，注意缩进和花括号
    • 三目运算符：条件 ? 值1 : 值2，简洁的条件表达式
    cpp
    #include <iostream>
using namespace std;

int main() {
    int score;
    cout << "输入成绩: ";
    cin >> score;

    if (score >= 90) {
        cout << "优秀" << endl;
    } else if (score >= 80) {
        cout << "良好" << endl;
    } else if (score >= 60) {
        cout << "及格" << endl;
    } else {
        cout << "不及格" << endl;
    }

    // 三目运算符
    string result = (score >= 60) ? "通过" : "未通过";
    cout << result << endl;
    return 0;
}

    ❌ 常见错误

    在if后加分号：if (x > 0); 空语句，无论条件如何都不执行任何操作，后面的代码块会无条件执行。

    12. switch语句

    switch语句根据表达式的值跳转到匹配的case分支，适合多值判断。

    🌟 核心知识点

    • 基本语法：switch(表达式) { case 值: 语句; break; ... default: 语句; }
    • break的作用：跳出switch，否则会穿透到下一个case（fall-through）
    • 表达式限制：必须是整数类型或枚举类型
    • C++17特性：[[fallthrough]]属性显式标记有意穿透
    cpp
    #include <iostream>
using namespace std;

int main() {
    int day;
    cout << "输入星期(1-7): ";
    cin >> day;

    switch (day) {
        case 1: cout << "星期一" << endl; break;
        case 2: cout << "星期二" << endl; break;
        case 3: cout << "星期三" << endl; break;
        case 6:
        case 7: cout << "周末" << endl; break;
        default: cout << "无效输入" << endl; break;
    }
    return 0;
}

    ❌ 常见错误

    忘记写break导致fall-through：case 1: cout << "一"; case 2: cout << "二"; 输入1时会输出"一二"而非"一"。

    13. for循环

    for循环是最常用的循环结构，适合已知循环次数的场景。

    🌟 核心知识点

    • 经典for：for(初始化; 条件; 更新) { 循环体 }
    • 范围for（C++11）：for(类型 变量 : 容器) { 循环体 }，遍历容器元素
    • 无限循环：for(;;) 或 for(;true;)
    • 多变量控制：for(int i=0, j=10; i<j; i++, j--)
    cpp
    #include <iostream>
#include <vector>
using namespace std;

int main() {
    // 经典for循环
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
        sum += i;
    }
    cout << "1+2+...+100 = " << sum << endl;

    // 范围for循环(C++11)
    vector<int> nums = {1, 2, 3, 4, 5};
    for (int n : nums) {
        cout << n << " ";
    }
    cout << endl;
    return 0;
}

    ❌ 常见错误

    循环条件写错导致死循环：for(int i=0; i<10; i--) 中i递减但判断i<10，i永远满足条件。应为 i++。

    14. while与do-while

    while循环适合未知循环次数的场景，do-while保证至少执行一次。

    🌟 核心知识点

    • while：while(条件) { 循环体 }，先判断后执行
    • do-while：do { 循环体 } while(条件);，先执行后判断，至少执行一次
    • 选择依据：已知次数用for，未知次数用while，至少一次用do-while
    cpp
    #include <iostream>
using namespace std;

int main() {
    // while: 计算数字位数
    int num = 12345, count = 0, temp = num;
    while (temp > 0) {
        temp /= 10;
        count++;
    }
    cout << num << " 有 " << count << " 位" << endl;

    // do-while: 菜单选择
    int choice;
    do {
        cout << "1.开始 2.设置 0.退出" << endl;
        cout << "请选择: ";
        cin >> choice;
    } while (choice != 0);
    return 0;
}

    ❌ 常见错误

    do-while末尾忘记分号：do { ... } while(cond) 缺少最后的 ;，会导致编译错误。

    15. break与continue

    break和continue用于改变循环的正常执行流程。

    🌟 核心知识点

    • break：立即跳出当前循环或switch语句
    • continue：跳过本次循环的剩余语句，直接进入下一次迭代
    • 嵌套循环：break和continue只影响最内层的循环
    cpp
    #include <iostream>
using namespace std;

int main() {
    // break: 找到第一个能被7整除的数
    for (int i = 1; i <= 100; i++) {
        if (i % 7 == 0) {
            cout << "第一个: " << i << endl;
            break;
        }
    }

    // continue: 输出1-10中的奇数
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) continue;
        cout << i << " ";
    }
    cout << endl;
    return 0;
}

    ❌ 常见错误

    在嵌套循环中期望break跳出外层循环，但break只能跳出最内层。如需跳出外层，可使用标志变量或goto。

    16. 一维数组

    数组是存储相同类型元素的固定大小连续内存空间。

    🌟 核心知识点

    • 声明与初始化：int arr[5];、int arr[] = {1,2,3};
    • 下标访问：从0开始，arr[0]是第一个元素
    • 数组大小：必须在编译期确定（C风格），变长数组不是标准C++
    • 越界访问：不检查边界，越界访问是未定义行为
    cpp
    #include <iostream>
using namespace std;

int main() {
    int scores[] = {85, 92, 78, 95, 88};
    int n = sizeof(scores) / sizeof(scores[0]);

    int sum = 0;
    for (int i = 0; i < n; i++) {
        sum += scores[i];
    }
    double avg = static_cast<double>(sum) / n;
    cout << "平均分: " << avg << endl;
    return 0;
}

    ❌ 常见错误

    数组越界访问：int arr[5]; arr[5] = 10; 下标5已越界（有效范围0-4），可能导致程序崩溃或数据损坏。

    17. 多维数组

    多维数组是数组的数组，最常用的是二维数组（矩阵）。

    🌟 核心知识点

    • 声明：int matrix[3][4]; 3行4列的二维数组
    • 初始化：int m[2][3] = {{1,2,3},{4,5,6}};
    • 内存布局：按行优先存储，连续内存
    • 作为函数参数：必须指定除第一维外的所有维度
    cpp
    #include <iostream>
using namespace std;

int main() {
    int matrix[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    21. 函数重载

    函数重载允许同名函数具有不同参数列表，编译器根据调用时的参数选择匹配版本。

    🌟 核心知识点

    • 重载条件：同名函数的参数类型、个数或顺序不同
    • 返回类型不参与重载：仅返回类型不同的同名函数不能重载
    • 名称修饰：编译器内部为重载函数生成唯一名称（name mangling）
    • 重载解析：精确匹配 → 类型提升匹配 → 标准转换匹配 → 用户定义转换匹配
    cpp
    #include <iostream>
using namespace std;

int add(int a, int b) { return a + b; }
double add(double a, double b) { return a + b; }
int add(int a, int b, int c) { return a + b + c; }

int main() {
    cout << add(1, 2) << endl;       // 调用int版本
    cout << add(1.5, 2.5) << endl;    // 调用double版本
    cout << add(1, 2, 3) << endl;     // 调用三参数版本
    return 0;
}

    ❌ 常见错误

    仅返回类型不同的重载：int foo(int x); 和 double foo(int x); 编译错误——参数列表必须不同。

    23. 内联函数

    内联函数建议编译器将函数体直接嵌入调用处，避免函数调用开销。

    🌟 核心知识点

    • 定义：inline int square(int x) { return x * x; }
    • 优势：消除函数调用开销（压栈、跳转、返回），适合短小频繁调用的函数
    • 局限性：inline只是建议，编译器可能忽略；递归函数、含循环的函数通常不会被内联
    • 与宏对比：inline有类型检查、不会产生副作用，比#define安全

    cout << "max(3, 7) = " << max(3, 7) << endl;
    return 0;
}

    ❌ 常见错误

    用宏实现"内联"：#define SQUARE(x) x*x，调用 SQUARE(1+2) 展开为 1+2*1+2=5而非9。应使用inline函数。

    24. 指针基础

    指针是存储内存地址的变量，是C++最强大也最危险的特性之一。

    🌟 核心知识点

    • 声明与初始化：int *p = &x; p存储x的地址，&是取地址运算符
    • 解引用：*p 访问指针指向的值，*是解引用运算符
    • 空指针：nullptr（C++11），不指向任何对象
    • 指针与地址：每个变量都有内存地址，指针用来间接访问变量
    cpp
    #include <iostream>
using namespace std;

int main() {
    int x = 42;
    int *p = &x;  // p指向x

    cout << "x的值: " << x << endl;
    cout << "x的地址: " << p << endl;
    cout << "通过指针访问: " << *p << endl;

    *p = 100;  // 通过指针修改x
    cout << "修改后x = " << x << endl;

    int *q = nullptr;  // 空指针
    cout << "空指针: " << q << endl;
    return 0;
}

    ❌ 常见错误

    使用未初始化的指针：int *p; *p = 10; p指向随机地址，写入该地址是未定义行为，可能崩溃。应先初始化：int x; int *p = &x;

    25. 指针与数组

    数组名可以当作指针使用，指针运算与数组下标访问等价。

    🌟 核心知识点

    • 数组名即指针：数组名是首元素地址，arr 等价于 &arr[0]
    • 指针运算：p+1 指向下一个元素（偏移sizeof(*p)字节）
    • 下标与指针等价：arr[i] 等价于 *(arr+i)
    • 指针遍历数组：用指针加减运算遍历数组元素
    cpp
    #include <iostream>
using namespace std;

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *p = arr;  // 指向数组首元素

    // 指针遍历
    for (int i = 0; i < 5; i++) {
        cout << *(p + i) << " ";
    }
    cout << endl;

    // arr[i] 与 *(arr+i) 等价
    cout << arr[2] << " " << *(arr + 2) << endl;
    return 0;
}

    ❌ 常见错误

    数组名是常量指针不能修改：int arr[5]; arr++; 编译错误，arr不能自增。可以用 int *p = arr; p++; 代替。

    26. 引用

    引用是已存在变量的别名，必须初始化且不能重新绑定。

    🌟 核心知识点

    • 声明：int &r = x; r是x的引用，对r的操作就是对x的操作
    • 必须初始化：引用声明时必须绑定到一个对象
    • 不可重绑定：引用一旦绑定就不能改为引用其他对象
    • 引用传参：函数参数使用引用可以避免拷贝并允许修改原变量
    • const引用：const int &r = x; 不能通过r修改x，可以绑定到右值

    cout << "交换前: " << x << " " << y << endl;
    swap(x, y);
    cout << "交换后: " << x << " " << y << endl;
    return 0;
}

    28. 构造与析构函数

    构造函数初始化对象，析构函数清理资源，两者是类生命周期管理的关键。

    🌟 核心知识点

    29. 继承

    继承允许从已有类派生新类，实现代码复用和层次化设计。

    🌟 核心知识点

    • 语法：class Derived : public Base { };

    • 构造顺序：先构造基类部分，再构造派生类部分；析构顺序相反

    30. 多态与虚函数

    多态允许通过基类接口操作不同派生类对象，虚函数是实现运行时多态的机制。

    🌟 核心知识点

    • 虚函数：virtual void foo(); 声明为虚函数，支持动态绑定
    • 纯虚函数：virtual void foo() = 0; 使类成为抽象类
    • override：C++11关键字，显式标记重写基类虚函数，防止拼写错误
    • 虚析构函数：基类析构函数应声明为virtual，确保delete基类指针时正确调用派生类析构函数
    cpp
    #include <iostream>
using namespace std;

class Shape {
public:
    virtual double area() = 0;  // 纯虚函数
    virtual ~Shape() {}         // 虚析构
};

class Circle : public Shape {
    double r;
public:
    Circle(double radius) : r(radius) {}
    double area() override { return 3.14159 * r * r; }
};

class Rectangle : public Shape {
    double w, h;
public:
    Rectangle(double w, double h) : w(w), h(h) {}
    double area() override { return w * h; }
};

int main() {
    Shape *s1 = new Circle(5);
    Shape *s2 = new Rectangle(4, 6);
    cout << "圆面积: " << s1->area() << endl;
    cout << "矩形面积: " << s2->area() << endl;
    delete s1; delete s2;
    return 0;
}

    ❌ 常见错误

    基类析构函数非virtual：Base *p = new Derived; delete p; 只调用Base析构，Derived析构不调用，资源泄漏。修正：声明 virtual ~Base() {}

    31. 模板

    • 模板实例化：编译器根据使用时的具体类型生成对应代码

    33. STL算法

    STL算法库提供大量通用算法，配合迭代器操作容器。

    🌟 核心知识点

int main() {
    vector<int> v = {5, 2, 8, 1, 9, 3};

    sort(v.begin(), v.end());  // 排序
    for (int x : v) cout << x << " ";
    cout << endl;

    auto it = find(v.begin(), v.end(), 8);
    if (it != v.end()) cout << "找到: " << *it << endl;

    int sum = accumulate(v.begin(), v.end(), 0);
    cout << "总和: " << sum << endl;

    reverse(v.begin(), v.end());
    for (int x : v) cout << x << " ";
    cout << endl;
    return 0;
}

    ❌ 常见错误

    对未排序容器使用binary_search：vector<int> v={3,1,4}; binary_search(v.begin(),v.end(),1); 可能返回false。binary_search要求容器已排序。

    34. 文件IO

    C++通过文件流类实现文件的读写操作。

    🌟 核心知识点

    • 文件流类：ofstream（写）、ifstream（读）、fstream（读写），需<fstream>
    • 打开模式：ios::in、ios::out、ios::app、ios::binary等
    • 写入文件：ofstream fout("file.txt"); fout << data;
    • 读取文件：逐行 getline(fin, line) 或逐词 fin >> word
    cpp
    #include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main() {
    // 写入文件
    ofstream fout("test.txt");
    fout << "Hello, File IO!" << endl;
    fout << "C++ is great!" << endl;
    fout.close();

    // 读取文件
    ifstream fin("test.txt");
    string line;
    while (getline(fin, line)) {
        cout << line << endl;
    }
    fin.close();
    return 0;
}

    ❌ 常见错误

    不检查文件是否成功打开：ifstream fin("nofile.txt"); 若文件不存在，后续读取操作静默失败。应检查：if (!fin) { cerr << "无法打开文件"; }

    35. 综合实战

    通过一个综合项目——学生成绩管理系统，综合运用前面所学的所有知识点。

    🌟 核心知识点

    • 类设计：Student类封装学生信息，GradeManager类管理所有学生
    • STL容器：使用vector存储学生数据，map存储科目成绩
    • 文件IO：将数据保存到文件，程序启动时加载
    • 多态应用：不同排序策略（按姓名、按成绩）可使用策略模式
    cpp
    #include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

class Student {
private:
    string name;
    vector<double> scores;
public:
    Student(string n) : name(n) {}
    void addScore(double s) { scores.push_back(s); }
    double getAverage() const {
        if (scores.empty()) return 0;
        double sum = 0;
        for (double s : scores) sum += s;
        return sum / scores.size();
    }
    string getName() const { return name; }
    void display() const {
        cout << name << " - 平均分: " << getAverage() << endl;
    }
};

class GradeManager {
private:
    vector<Student> students;
public:
    void addStudent(const Student& s) { students.push_back(s); }
    void showAll() {
        for (const auto& s : students) s.display();
    }
    void sortByAvg() {
        sort(students.begin(), students.end(),
            [](const Student& a, const Student& b) {
                return a.getAverage() > b.getAverage();
            });
    }
};

int main() {
    GradeManager mgr;
    Student s1("张三"); s1.addScore(85); s1.addScore(90); s1.addScore(78);
    Student s2("李四"); s2.addScore(92); s2.addScore(88); s2.addScore(95);
    mgr.addStudent(s1); mgr.addStudent(s2);
    mgr.sortByAvg();
    mgr.showAll();
    return 0;
}

    ❌ 常见错误

    lambda捕获未考虑生命周期：lambda按引用捕获局部变量，如果lambda在变量销毁后执行，会产生悬垂引用。优先按值捕获。
